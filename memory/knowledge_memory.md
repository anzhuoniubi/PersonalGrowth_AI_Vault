## ⭐ 核心定位知识（2026-07-21 新增）

### 「半醒之间」IP系统
- **核心命题**：AI时代个人价值重构
- **价值公式变化**：人的价值从「学历×职业×组织身份」→「认知×创造力×表达力×AI杠杆」
- **三层内容体系**：觉察层(40%) → 重构层(30%) → 创造层(30%)
- **长期主线**：一部连续剧——一个普通人如何在AI时代完成一次个人价值重构实验
- **90天实验计划**：阶段1找到共鸣 → 阶段2形成系列 → 阶段3强化行动

### 关键表达原则
- 先解决痛苦，再升维
- 从"明天还要上班的人"的视角出发
- 不是"我教你"，是"我在做，你看"
