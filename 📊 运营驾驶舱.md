## 📈 核心指标

```dataview
TABLE 
  length(rows) AS "已发布",
  sum(rows.曝光) AS "总曝光",
  sum(rows.互动) AS "总互动",
  round(sum(rows.互动) / sum(rows.阅读) * 100, 1) + "%" AS "互动率"
FROM "05_内容生产库"
WHERE status = "已发布"
FLATTEN 0 AS 曝光
FLATTEN 0 AS 互动
FLATTEN 0 AS 阅读
GROUP BY true
```
