## 一、IP总定位

### 名称
**半醒之间**

### 一句话定位
> 一个普通人在AI时代，通过觉察自己、利用AI、建立个人品牌，探索个人价值重构的长期实践。

### 电梯演讲
> "当AI能写能画能做大部分事情，人最深的焦虑不是'我会不会被替代'，而是'我还有什么价值'。
> 我帮那些习惯深想的年轻人，把这个问题从焦虑变成燃料。
> 不是我教你什么，是你本来就有的思考优势——只是还没找到它的战场。
> 关注我，从焦虑的收集者变成自洽的建造者。"

---

## 二、核心世界观

### 核心命题
> **AI时代个人价值重构**

你关注的不是"如何使用AI"，而是：
> 当AI改变社会生产方式后，一个普通人的价值应该如何重新定义？

### 价值公式变化

| 过去 | 未来 |
|------|------|
| 人的价值 = 学历 × 职业 × 组织身份 | 人的价值 = 认知 × 创造力 × 表达力 × AI杠杆 |
| 什么大学/什么公司/什么职位 | 建立影响力/创造内容/形成个人品牌/创造新的职业可能 |

## 三、IP人格

### 表层身份
AI时代个人品牌实践者

### 内核身份
一个曾经被传统路径定义的人，正在重新寻找自己的价值

### 人设关键词

| 不是 | 而是 |
|-----|------|
| ❌ AI专家 | ✅ 探索者 |
| ❌ 成功导师 | ✅ 实践者 |
| ❌ 财富导师 | ✅ 创造者 |
| ❌ 觉醒导师 | ✅ 实验者 |

### 一句话人设
> "我不是告诉你答案的人，我是正在验证另一种人生可能性的人。"

## 四、核心用户

### 一句话画像
> 被传统路径压抑，但相信自己还有可能的人。

### 详细画像

| 维度 | 描述 |
|------|------|
| 年龄 | 25-35岁 |
| 背景 | 普通家庭出身，接受传统教育 |
| 现状 | 有稳定工作但缺乏意义感，工作只是谋生手段 |
| 困惑 | 在AI浪潮中感到冲击，不知道自己的人生方向 |
| 特征 | 有审美、表达欲、思考欲，感觉自己"还有东西没有被发挥出来" |
| 工作状态 | 工作几年了，但不作为终身想要的工作 |

## 五、三层内容体系

### 内容漏斗

```
┌─────────────────────────────────────┐
│        第一层：觉察（40%）            │
│   "我为什么活得不像自己？"           │
│   吸引同类，建立情感连接              │
├─────────────────────────────────────┤
│        第二层：重构（30%）            │
│   "AI时代，我这种人还有机会吗？"      │
│   形成差异化，提供新视角              │
├─────────────────────────────────────┤
│        第三层：创造（30%）            │
│   "如何把自己变成价值？"             │
│   建立信任，形成商业闭环              │
└─────────────────────────────────────┘
```

### 第一层：觉察（吸引同类）

| 内容方向 | 示例标题 |
|---------|---------|
| 社会时钟觉醒 | 《毕业后才发现，我们努力追求的稳定，可能不是自己想要的人生》 |
| 原生家庭 | 《很多人的人生，其实是在完成父母未完成的期待》 |
| 高敏感人格 | 《高敏感的人，不是真的脆弱，而是感知能力太强》 |
| 普通人的隐藏天赋 | 《为什么有些人看似普通，却总觉得自己还有另一种可能》 |

### 第二层：重构（形成差异）

| 内容方向 | 示例标题 |
|---------|---------|
| AI与人的关系 | 《AI出现后，我第一次觉得普通人可能拥有新的机会》 |
| AI时代能力变化 | 《未来最值钱的能力，也许不是执行力》 |
| AI帮助个人成长 | 《一个没有资源的普通人，如何利用AI放大自己》 |

### 第三层：创造（建立信任）

| 内容方向 | 示例标题 |
|---------|---------|
| 个人品牌实验 | 《从0开始打造个人品牌第1天》 |
| AI系统搭建 | 《一个人的AI团队，我是怎么搭建的》 |

## 六、IP故事线

```
第一幕：被定义
  普通家庭出生 → 接受传统教育 → 按照社会时钟成长
      ↓
第二幕：产生裂缝
  稳定 ≠ 幸福 → 成功模板 ≠ 自我价值
      ↓
第三幕：觉醒
  我是谁？ → 我的优势是什么？ → 我的人生可能性在哪里？
      ↓
第四幕：AI时代机会
  AI第一次让普通人的思考/审美/创造/表达可以被放大
      ↓
第五幕：个人品牌实验
  从0建立表达体系 → 创造内容 → 形成影响力 → 验证个人价值
```

---

## 七、内容红线

- ❌ 不制造焦虑
- ❌ 不承诺具体结果
- ❌ 不抄袭或洗稿
- ❌ 不站在山顶说话（用户还在山脚）
- ✅ 所有内容先问："这个观点，如何帮助一个明天早上还要上班的人？"

### 表达原则
> 不要："AI时代，人类价值正在重构。"
> 而是："如果你的工作正在消耗你，AI可能第一次给普通人一个重新选择人生的机会。"
> **先解决痛苦，再升维。**

## 八、商业化路径（远期）

```
内容影响力 → 个人品牌 → 知识产品 → AI个人品牌咨询/训练 → 一人公司生态
```

> **当前阶段（0-10000粉）：只关注建立三个认知**
> - 用户："这个人在探索我也关心的问题。"
> - 平台："这个账号属于AI时代个人成长领域。"
> - 自己："哪些表达能够产生共鸣？"

## 九、90天实验计划

### 阶段1（1-30天）
- 目标：找到用户共鸣
- 发布：30篇
- 比例：70%觉察 + 20%AI时代 + 10%个人记录

### 阶段2（31-60天）
- 识别高互动主题
- 形成系列内容

### 阶段3（61-90天）
- 强化"AI + 个人品牌实践"
- 让用户知道：你不仅思考，你在行动
