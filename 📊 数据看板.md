## 📝 最近发布内容

```dataview
TABLE date AS "日期", status AS "状态", rating AS "评级"
FROM "05_内容生产库"
WHERE status = "已发布" AND date != null
SORT date DESC
LIMIT 5
```

## 📋 选题状态

```dataview
TABLE priority AS "优先级", status AS "状态", source AS "来源"
FROM "03_选题库"
WHERE status != null AND status != "" AND file.name != "选题看板" AND file.name != "选题卡片模板"
SORT priority DESC
LIMIT 10
```
