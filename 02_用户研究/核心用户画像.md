## 详细画像

### 基础属性

| 维度 | 描述 |
|------|------|
| 年龄 | 25-35岁 |
| 性别 | 不限（前期以男性为主） |
| 家庭背景 | 普通家庭出身，非富非贵 |
| 教育 | 本科及以上，接受过传统教育体系 |
| 城市 | 一二线城市为主 |
| 职业 | 互联网/金融/制造业白领，自由职业者 |
| 收入 | 15k-40k/月 |

### 生活状态

- 工作3-8年，已经过了职场新鲜期
- 工作只是谋生手段，不是终身志业
- 有稳定收入，但缺乏意义感和成就感
- 下班后的时间不知道如何有效利用
- 在AI浪潮中感到焦虑：自己的技能是否会被替代

### 心理特征

- **有表达的欲望**：内心有很多想法，但不知道怎么系统表达
- **有审美的追求**：对内容的质量和美感有要求，不满足于快餐信息
- **有思考的习惯**：习惯深想问题，但容易陷入过度思考
- **有不甘的心态**：感觉自己"还有东西没有被发挥出来"
- **有改变的愿望**：但缺乏系统的方法和行动路径

### 核心痛点

| 痛点层级 | 具体表现 |
|---------|---------|
| 表层 | 工作没意义、AI焦虑、不知道未来方向 |
| 中层 | 想改变但没方法、收藏了太多从未执行、自我怀疑 |
| 深层 | 被传统成功模板定义太久，不知道自己真正想要什么 |

### 隐性需求

1. **被理解**：有人能说出他们说不清的困惑
2. **新路径**：看到另一种人生可能性的真实案例
3. **可行性**：有可执行的步骤，不是鸡汤
4. **陪伴感**：有人在走同样的路，减少孤独感
5. **希望感**：在不确定性中找到方向感

## 用户分层

### 🎯 核心用户（70%内容服务于此）
**「觉醒中的探索者」**
- 正在经历从"被定义"到"寻找自我"的转变期
- 对"社会时钟""原生家庭""自我价值"有强烈共鸣
- 看到AI相关内容会思考"这对我意味着什么"
- 行为特征：深夜活跃、评论偏长篇自我暴露、收藏决策型内容
- **已验证信号**：身份标签+时代困惑=高互动

### 📌 次级用户（25%内容尝试覆盖）
**「焦虑中的观望者」**
- 有类似困境但尚未进入觉醒阶段
- 被"觉察层"内容吸引，但更需要方法论牵引
- 触达路径：被共鸣内容推荐到→如果看到可执行内容会关注
- 行为特征：收藏＞点赞＞评论

### 🔄 外围用户（5%内容触及）
**「路过的好奇者」**
- 偶然刷到，被标题或封面吸引
- 贡献点赞但不关注
- 价值：拉高曝光基数

## 用户与IP的关系期望

用户期待的"半醒之间"：
> 一个比我先行一步的普通人，正在探索一条我不敢尝试的路。
> 他不是导师，不是专家，而是一个**真实的实验者**。
> 我看他的内容，是因为我想看到"另一种可能性"真的在发生。
