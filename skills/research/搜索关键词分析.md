# 搜索关键词分析 Skill

> 验证来源：SEO行业实践 / 小红书搜索算法 / 长尾关键词理论（Chris Anderson）
> 最后更新：2026-07-17

---

## 1. Skill名称
搜索关键词分析

## 2. 使用场景
- 选题阶段：验证选题的搜索量和竞争度
- 标题阶段：在标题中嵌入高搜索关键词
- 标签阶段：选择搜索权重高的标签
- 系列内容规划：围绕核心关键词做内容矩阵

## 3. 输入格式
```yaml
topic: ""
niche: "个人成长" | "AI时代" | "自我觉察" | "个人品牌"
target_keywords: []
output_type: "选题验证" | "标题关键词" | "标签策略"
```

## 4. 核心框架

### 框架A：关键词四象限（选择赛道）

> 来源：SEO行业经典分析框架
> 适用：判断一个关键词是否值得做

```
              搜索量
                ↑
         高搜索·低竞争    高搜索·高竞争
    🟢      蓝海      ←→      红海
    最佳切入点          头部竞争激烈
                |
    🔵      冷门      🟡    鸡肋
         低搜索·低竞争    低搜索·高竞争
    可以做但流量有限     做了也没人搜
                ↓
```

| 象限 | 结论 | 内容策略 |
|------|------|---------|
| 🟢 高搜索·低竞争 | **首选** 蓝海关键词 | 全力做，抢排名 |
| 🔴 高搜索·高竞争 | **慎入** 红海关键词 | 除非有独特角度否则不做 |
| 🔵 低搜索·低竞争 | **备选** 冷门关键词 | 可做长尾组合，但别当主力 |
| 🟡 低搜索·高竞争 | **放弃** 鸡肋关键词 | 做了没流量，直接跳过 |

**小红书实操法：**
在小红书搜索框敲关键词，看：
- **搜索结果数** → 判断竞争度（<1万低竞争，1-10万中，>10万高）
- **顶部笔记的点赞数** → 判断头部强度（顶部都是千赞=高竞争）
- **搜索下拉词** → 判断用户搜索习惯和长尾机会

---

### 框架B：搜索意图分类

> 来源：Google搜索质量评估指南（E-E-A-T）
> 适用：判断用户搜这个词时到底想要什么

| 搜索意图 | 定义 | 小红书示例 | 内容类型匹配 |
|---------|------|-----------|-------------|
| 🧠 求知型 | 想学东西 | "AI时代普通人还有什么机会""什么是个人价值重构" | 科普/概念解释 |
| ❓ 问题型 | 想解决问题 | "工作没意义怎么办""如何找到自己的人生方向" | 方法论/清单 |
| 🛍️ 决策型 | 想买东西 | "普通人适合做什么副业""个人品牌怎么开始" | 评测/对比/推荐 |
| 👀 好奇型 | 随便看看 | "普通人的一天""30岁觉醒的真实故事" | 娱乐/共鸣/故事 |

**发什么内容取决于意图：**
- 同一关键词"工作没意义怎么办"，可以按意图出3篇不同内容
- 求知型 → "为什么工作会让你觉得没意义（从心理学解释）"
- 问题型 → "工作没意义时的3个自救方法"
- 好奇型 → "30岁重新开始的真实经历"

---

### 框架C：长尾关键词矩阵

> 来源：Chris Anderson《长尾理论》
> 适用：用多个低竞争长尾词组合获取稳定流量

**公式：** `{核心词} + {场景/人群/痛点/方案}`

```
核心词：工作没意义
         ↓
   ┌─────┼─────┐
 场景词  人群词  方案词
  │      │      │
 深夜    职场人  三步
 工作    年轻白领  笔记
 转型   年轻白领  认知
```

**实操路径：**
1. 选定1个核心词（高搜索+中高竞争）
2. 组合3-5个长尾词（低竞争）
3. 每篇内容锁定1个长尾词为主关键词
4. 覆盖全部长尾词后，自然获取核心词排名

---

## 5. 执行流程

### Step 1：核心词提取
从选题或内容中提取1-2个核心关键词。

### Step 2：搜索量验证
在小红书搜索框输入核心词，记录：
- 搜索结果数
- 顶部笔记的平均点赞区间
- 下拉词列表

### Step 3：四象限定位
把核心词放入四象限，判断是否值得做。

### Step 4：搜索意图匹配
确定核心词的搜索意图类型，决定内容形式。

### Step 5：长尾词矩阵
围绕核心词生成3-5个长尾词组合。

### Step 6：标题嵌入
从长尾词矩阵中选择最佳词嵌入标题（前10字最佳）。


## 8. 避坑清单

- [ ] 不只看搜索量，还要看竞争度（高搜索高竞争=做了也不一定有流量）
- [ ] 标题前10字必须包含核心关键词
- [ ] 不是所有高搜索关键词都适合品牌——要过品牌相关度这一关
- [ ] 搜索意图和内容形式要对齐（用户搜"怎么办"就别写科普）
- [ ] 长尾词策略需要持续3-5篇才能见效，不要做一篇就放弃
