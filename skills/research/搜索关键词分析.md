# 搜索关键词分析

## 使用场景

- 选题阶段：验证选题的搜索量和竞争度
- 标题阶段：在标题中嵌入高搜索关键词
- 标签阶段：选择搜索权重高的标签
- 系列内容规划：围绕核心关键词做内容矩阵

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| topic | string | 选题主题 |
| niche | string | 个人成长 / AI时代 / 自我觉察 / 个人品牌 |
| output_type | string | 选题验证 / 标题关键词 / 标签策略 |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| target_keywords | string[] | 目标关键词列表 |

## 执行流程

1. **核心词提取**：从选题或内容中提取 1-2 个核心关键词
2. **搜索量验证**：在小红书搜索框输入核心词，记录搜索结果数、顶部笔记点赞区间、下拉词列表
3. **四象限定位**：把核心词放入四象限，判断是否值得做
4. **搜索意图匹配**：确定核心词的搜索意图类型，决定内容形式
5. **长尾词矩阵**：围绕核心词生成 3-5 个长尾词组合
6. **标题嵌入**：从长尾词矩阵中选择最佳词嵌入标题（前 10 字最佳）

### 核心框架（流程内小节）

#### 框架 A：关键词四象限
```
              搜索量
                ↑
         高搜索·低竞争    高搜索·高竞争
    🟢      蓝海      ←→      红海
    最佳切入点          头部竞争激烈
                |
    🔵      冷门      🟡    鸡肋
         低搜索·低竞争    低搜索·高竞争
    可以做但流量有限     做了也没人搜
                ↓
```

| 象限 | 结论 | 内容策略 |
|------|------|---------|
| 🟢 高搜索·低竞争 | **首选** 蓝海关键词 | 全力做，抢排名 |
| 🔴 高搜索·高竞争 | **慎入** 红海关键词 | 除非有独特角度否则不做 |
| 🔵 低搜索·低竞争 | **备选** 冷门关键词 | 可做长尾组合，但别当主力 |
| 🟡 低搜索·高竞争 | **放弃** 鸡肋关键词 | 做了没流量，直接跳过 |

**小红书实操法：**
- 搜索结果数 <1 万 = 低竞争，1-10 万 = 中，>10 万 = 高
- 顶部笔记都是千赞 = 高竞争
- 搜索下拉词 → 判断用户搜索习惯和长尾机会

#### 框架 B：搜索意图分类
| 搜索意图 | 定义 | 小红书示例 | 内容类型匹配 |
|---------|------|-----------|-------------|
| 🧠 求知型 | 想学东西 | "AI时代普通人还有什么机会" | 科普/概念解释 |
| ❓ 问题型 | 想解决问题 | "工作没意义怎么办" | 方法论/清单 |
| 🛍️ 决策型 | 想买东西 | "普通人适合做什么副业" | 评测/对比/推荐 |
| 👀 好奇型 | 随便看看 | "普通人的一天" | 娱乐/共鸣/故事 |

#### 框架 C：长尾关键词矩阵
**公式**：`{核心词} + {场景/人群/痛点/方案}`

```
核心词：工作没意义
         ↓
   ┌─────┼─────┐
 场景词  人群词  方案词
  │      │      │
 深夜    职场人  三步
 工作    年轻白领  笔记
 转型   年轻白领  认知
```

### 判断标准（if-then）
- 如果四象限定位是红海，则必须找到独特角度或放弃。
- 如果搜索意图与内容形式不匹配，则必须调整内容形式。
- 如果长尾词组合 <3 个，则必须扩展关键词池。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 核心词 | string | — |
| 搜索结果数 | string | — |
| 头部笔记点赞 | string | 区间 |
| 四象限定位 | string | 🟢/🔴/🔵/🟡 |
| 主要搜索意图 | string | 求知/问题/决策/好奇 |
| 长尾词矩阵 | object[] | 词/位置/竞争度/优先级 |
| 标签策略 | string | — |

### 输出示例
```markdown
## 关键词分析报告

### 核心词：{}
- 搜索结果数：{N}
- 头部笔记点赞：{区间}
- 四象限定位：{🟢/🔴/🔵/🟡}
- 主要搜索意图：{求知/问题/决策/好奇}
- 结论：{做/不做/换个角度做}

### 长尾词矩阵
| 长尾词 | 建议标题位置 | 竞争度 | 优先级 |
|--------|------------|:-----:|:------:|
| {词1} | {标题/标签/正文} | 低/中/高 | ★★★ |
| {词2} | {标题/标签/正文} | 低/中/高 | ★★☆ |

### 标签策略
#{核心标签} #{长尾标签1} #{长尾标签2} #{领域标签}
```

## 示例

### 示例 1：选题关键词验证
**输入**：
```yaml
topic: "工作没意义，普通人如何在AI时代找到新方向"
niche: "个人成长"
target_keywords: ["工作没意义", "AI时代个人价值"]
output_type: "标题关键词"
```

**输出**：
```markdown
## 关键词分析报告

### 核心词：工作没意义
- 搜索结果数：约 2.3 万篇
- 头部笔记点赞：500-2000 赞
- 四象限定位：🟢 高搜索·低竞争（蓝海）
- 主要搜索意图：问题型（用户想找解决方案）
- 结论：✅ 值得做，竞争度适中

### 长尾词矩阵
| 长尾词 | 建议标题位置 | 竞争度 | 优先级 |
|--------|------------|:-----:|:------:|
| 工作没意义怎么办 | 标题 | 低 | ★★★ |
| 普通人AI时代机会 | 标题 | 低 | ★★★ |
| 个人价值重构 | 标题 | 低 | ★★★ |
| 人生方向迷茫 | 标签 | 中 | ★★☆ |
| AI时代自我定位 | 正文/标签 | 低 | ★★☆ |

### 标签策略
#工作没意义 #AI时代 #个人价值 #自我成长 #人生方向
```

### 避坑清单
- [ ] 不只看搜索量，还要看竞争度
- [ ] 标题前 10 字必须包含核心关键词
- [ ] 不是所有高搜索关键词都适合品牌
- [ ] 搜索意图和内容形式要对齐
- [ ] 长尾词策略需要持续 3-5 篇才能见效
