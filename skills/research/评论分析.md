# 评论分析

## 使用场景

- 每日评论区洞察
- 竞品评论区研究
- 内容反馈收集
- 用户情绪追踪
- 热门话题发现

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| comments | object[] | 评论列表 |
| source_note.type | string | own / competitor / industry |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| source_note.title | string | 笔记标题 |
| source_note.url | string | 笔记链接 |

### comments 字段
| 字段 | 类型 | 说明 |
|------|------|------|
| user | string | 用户名（匿名化） |
| content | string | 评论原文 |
| likes | number | 点赞数 |
| replies | string[] | 回复 |
| timestamp | string | 时间戳 |

## 执行流程

1. **评论收集**：按目标内容收集评论区
2. **分类标记**：按类型分类（问题/共鸣/争议/建议）
3. **高频词提取**：找到重复出现的词和表达
4. **情绪分析**：正面/负面/中性情绪分布
5. **需求提取**：用户评论中反映的潜在需求
6. **争议点识别**：什么话题引发讨论
7. **高价值评论标注**：有洞见的用户反馈
8. **洞察总结**：输出分析报告

### 判断标准（if-then）
- 如果某条评论点赞 ≥100，则必须纳入高价值评论分析。
- 如果某关键词出现 ≥3 次，则必须纳入高频词。
- 如果负面评论占比 ≥30%，则必须触发情绪预警。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 分析笔记 | string | 标题/链接 |
| 评论总数 | number | — |
| 高互动评论数 | number | — |
| 高频词 | string[] | top 10 |
| 情绪分布 | object | 正面/中性/负面占比 |
| 关键洞察 | string[] | — |
| 内容启发 | object | 用户想知道/困惑/创作建议 |

### 输出示例
```markdown
## 评论分析报告
### 基本信息
- 分析笔记：{}
- 评论总数：{}
- 高互动评论：{}

### 高频词云
{top 10 高频词}

### 情绪分布
- 正面：{百分比}
- 中性：{百分比}
- 负面：{百分比}

### 关键洞察
1. {洞察1}
2. {洞察2}

### 内容启发
- 用户想知道：{}
- 用户困惑：{}
- 创作建议：{}
```

## 示例

### 示例 1：自律内容评论区
**输入**：
```yaml
comments:
  - content: "每次看完这种自律内容就更焦虑了"
    likes: 245
  - content: "求方法，怎么开始"
    likes: 120
  - content: "试过，坚持不了三天"
    likes: 89
source_note:
  title: "如何坚持早起100天"
  type: "own"
```

**输出**：
```markdown
## 评论分析报告
### 基本信息
- 分析笔记：如何坚持早起100天
- 评论总数：247
- 高互动评论：15

### 高频词云
坚持、起不来、好难、求方法、试过、失败、佩服

### 情绪分布
- 正面：45%（敬佩、想尝试）
- 中性：35%（问方法、求细节）
- 负面：20%（挫败感、自我怀疑）

### 关键洞察
1. 用户最关心"如何坚持"而非"为什么要早起"
2. 失败经历共鸣最强

### 内容启发
- 用户想知道：有没有适合普通人的坚持方法
- 用户困惑：为什么自己总是坚持不下去
- 创作建议：创作"早起失败的3个真正原因"填补认知空白
```
