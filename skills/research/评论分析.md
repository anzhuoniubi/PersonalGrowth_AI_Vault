# 评论分析 Skill

## 1. Skill名称
评论分析

## 2. 使用场景
- 每日评论区洞察
- 竞品评论区研究
- 内容反馈收集
- 用户情绪追踪
- 热门话题发现

## 3. 输入格式
```yaml
comments:
  - user: ""
    content: ""
    likes: 0
    replies: []
    timestamp: ""
source_note:
  title: ""
  url: ""
  type: "own" | "competitor" | "industry"
```

## 4. 执行流程
1. **评论收集**：按目标内容收集评论区
2. **分类标记**：按类型分类（问题/共鸣/争议/建议）
3. **高频词提取**：找到重复出现的词和表达
4. **情绪分析**：正面/负面/中性情绪分布
5. **需求提取**：用户评论中反映的潜在需求
6. **争议点识别**：什么话题引发讨论
7. **高价值评论标注**：有洞见的用户反馈
8. **洞察总结**：输出分析报告

## 5. 输出格式
```markdown
## 评论分析报告
### 基本信息
- 分析笔记：{}
- 评论总数：{}
- 高互动评论：{}

### 高频词云
{top 10 高频词}

### 情绪分布
- 正面：{百分比}
- 中性：{百分比}
- 负面：{百分比}

### 关键洞察
1. {洞察1}
2. {洞察2}

### 内容启发
- 用户想知道：{}
- 用户困惑：{}
- 创作建议：{}
```

## 6. 示例
```markdown
## 评论分析报告
### 基本信息
- 分析笔记："如何坚持早起100天"
- 评论总数：247
- 高互动评论：15

### 高频词云
坚持、起不来、好难、求方法、试过、失败、佩服

### 情绪分布
- 正面：45%（敬佩、想尝试）
- 中性：35%（问方法、求细节）
- 负面：20%（挫败感、自我怀疑）

### 关键洞察
1. 用户最关心"如何坚持"而非"为什么要早起"
2. 失败经历共鸣最强

### 内容启发
- 创作"早起失败的3个真正原因"填补认知空白
```
