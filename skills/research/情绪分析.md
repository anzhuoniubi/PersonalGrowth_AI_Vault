# 情绪分析 Skill

## 1. Skill名称
情绪分析

## 2. 使用场景
- 内容情感基调设计
- 评论区情绪监测
- 选题情绪匹配
- 品牌情感定位

## 3. 输入格式
```yaml
text_content: ""
analysis_type: "comment" | "content" | "topic"
focus_dimensions:
  - "intensity"
  - "valence"
  - "trigger_words"
context: ""
```

## 4. 执行流程
1. **情绪识别**：识别文本中的情绪标签
2. **强度评估**：情绪强烈程度（1-10）
3. **正负判断**：积极/消极/中性
4. **触发词提取**：引发情绪的关键词
5. **情绪归因**：情绪背后的原因
6. **内容适配**：根据情绪设计内容基调
7. **共鸣点匹配**：找到情绪与内容的连接点

## 5. 输出格式
```markdown
## 情绪分析结果
### 情绪画像
- 主导情绪：{}
- 情绪强度：{}
- 情绪基调：{积极/消极/中性}

### 触发因素
- 触发电词：{}
- 情绪来源：{}

### 内容策略
- 匹配情绪：{}
- 建议语气：{}
- 避免雷区：{}
- 情感共鸣点：{}
```

## 6. 示例
```markdown
## 情绪分析结果
### 情绪画像
- 主导情绪：焦虑+希望（混合）
- 情绪强度：7/10
- 情绪基调：积极焦虑（想变好但不知如何开始）

### 触发因素
- 触发词：30岁、瓶颈、落后、同龄人
- 情绪来源：社会比较+年龄焦虑

### 内容策略
- 匹配情绪：先用共情化解焦虑，再给解决方案
- 建议语气：温和坚定，不制造更多焦虑
- 避免雷区：不要用"再不xxx就晚了"的恐吓式表达
- 情感共鸣点：承认焦虑是正常的，给出可控的路径
```
