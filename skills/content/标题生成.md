# 标题生成 Skill

## 1. Skill名称
标题生成

## 2. 使用场景
- 每篇内容发布前的标题设计
- A/B测试多个标题方案
- 封面文案设计
- 标题优化迭代

## 3. 输入格式
```yaml
content_topic: ""
content_type: "personal_story" | "methodology" | "review" | "reflection"
target_audience: ""
core_message: ""
emotion_tone: ""
key_words: []
constraints:
  max_length: 20
  style: "xiaohongshu"
```

## 4. 核心规则（先读）

### 字数上限
> 小红书标题限制**20字以内**。超过20字的标题不会被完整展示，点击率大幅下降。

### 前10字法则
> 标题前10字占点击率权重的**60%以上**。核心关键词、情绪词、数字必须在10字内出现。

### 三个必须
- ✅ **必须**有情绪钩子（痛点/疑问/反常识）
- ✅ **必须**有目标人群指向（或话题标签）
- ✅ **必须**与封面、正文内容一致（不一致会被降权）

## 5. 四大黄金模板（来源：小红书KOL推广验证）

### 模板A：数字具象法
**公式**：`{具体数字} + {痛点场景} + {解决方案/结果承诺}`
**市场数据**：带数字的标题CTR高**47%**，奇数比偶数CTR高**11-18%**
**适用**：方法论类、清单类、干货类

| 要素 | 说明 | 示例 |
|------|------|------|
| 数字 | 用奇数，≤10，越具体越好 | "3个""7天""30岁" |
| 痛点 | 来自评论区真实高频词 | "内耗""无效努力" |
| 方案/结果 | 可感知的变化 | "停止内耗""提升3倍" |

```
"3个习惯帮我走出内耗"
"停止无效努力后，我的成长快了3倍"
"30岁才明白的5个人生道理"
```

---

### 模板B：身份共鸣法
**公式**：`{人群标签/身份} + {具体场景} + {利益点}`
**市场数据**：含身份标签的标题精准触达率提升显著，互动率高**40%**
**适用**：觉察层内容、情绪共鸣类、人群定位类

| 要素 | 说明 | 示例 |
|------|------|------|
| 人群标签 | 让目标用户觉得"这就是我" | "普通家庭""习惯深想""不甘平庸的人" |
| 场景 | 具体的处境或困境 | "不被选择""择偶中""深夜" |
| 利益点 | 看完能获得什么 | "方法""答案""出路" |

```
"一个习惯深想的普通人，在AI时代能找到自己的位置吗"
"高敏感的人，最需要停止向内看"
"内向的人，不需要逼自己变外向"
```

---

### 模板C：悬念反差法
**公式**：`{反常识认知} + {问号/转折/留白}`
**市场数据**：身份反差的标题互动率提升约**40%**
**适用**：认知反转类、观点输出类、打破迷思类

| 要素 | 说明 | 示例 |
|------|------|------|
| 常识反转 | 跟多数人的认知反着来 | "关于XX，90%的人都理解错了" |
| 转折词 | "但""却""其实""别再" | "别再XX了""但其实" |
| 留白 | 不说完，让人想点开 | 结尾用问号或省略号 |

```
"关于自律，90%的人都理解错了"
"别再'向内求'了"
"那股力量不该被廉价地兑换掉"
```

---

### 模板D：情绪对比法
**公式**：`{情绪词} + {对比/反差} + {关键词}`
**市场数据**：含情绪词的标题打开率高**23%**
**适用**：情绪共鸣类、成长叙事类、前后对比类

| 要素 | 说明 | 示例 |
|------|------|------|
| 情绪词 | 引发情绪共鸣 | "焦虑""内耗""累""自由" |
| 对比 | 前后状态/认知的差距 | "从A到B""不是……而是" |
| 关键词 | 话题核心词 | "成长""自我""选择" |

```
"从焦虑到自洽，我只做了一件事"
"不是不够好，是体系不适合你"
"蝴蝶不需要排行榜"
```

## 6. 执行流程

1. **内容核心提取**：从正文中提取最引人注目的点
2. **模板匹配**：根据内容类型选择1-2个模板

| 内容类型 | 首选模板 | 次选模板 |
|---------|---------|---------|
| 方法论/干货 | 模板A 数字具象法 | 模板D 情绪对比法 |
| 情绪/内耗/觉察层 | 模板B 身份共鸣法 | 模板D 情绪对比法 |
| 认知反转/观点 | 模板C 悬念反差法 | 模板A 数字具象法 |
| 复合型 | 主模板+次模板组合 | — |

3. **生成3个变体**：每个选定模板出1-2个标题
4. **前10字检验**：核心信息是否在10字内出现
5. **字数检查**：全部控制在20字以内
6. **真实性验证**：是否与正文一致，不标题党
7. **优化排序**：按点击潜力输出Top 3

### 输出格式
```markdown
## 标题方案
### 首选（推荐指数：★★★★★）
{标题}（{所用模板}）

### 备选
1. {标题}（{所用模板}，★★★★☆）
2. {标题}（{所用模板}，★★★★☆）
3. {标题}（{所用模板}，★★★☆☆）

### 封面文案
{封面文字}
```

### 避坑清单
- [ ] 不超过20字
- [ ] 前10字包含核心信息
- [ ] 没有"最""第一""绝对"等极限词
- [ ] 没有"震惊体"
- [ ] 与封面和正文一致
- [ ] 不使用无法证伪的身份标签

## 5. 输出格式
```markdown
## 标题方案
### 首选（推荐指数：★★★★★）
{标题}

### 备选
1. {标题}（推荐指数：★★★★☆）
2. {标题}（推荐指数：★★★★☆）
3. {标题}（推荐指数：★★★☆☆）

### 封面文案
{封面文字}
```

## 6. 示例
### 输入
topic: "如何通过建立知识体系提升学习效率"
type: "methodology"

### 输出
```
## 首选
"建立个人知识体系后，我的学习效率提升了3倍"

## 备选
1. "关于知识管理，90%的人都理解错了"
2. "3个步骤搭建你的第二大脑"
3. "为什么你的知识越学越乱？"

## 封面文案
学习效率提升3倍的秘密
```
