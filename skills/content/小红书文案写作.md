# 小红书文案写作（xhs-note-creator）

> 基于 [Auto-Redbook-Skills](https://github.com/comeonzhj/Auto-Redbook-Skills) 渲染工具链
> 工具路径：[[skills/redbook-auto/README.md]]
> 对齐定位：[[01_个人定位/半醒之间IP定位]]

根据选题创作小红书笔记内容，自动生成精美图片卡片，可选发布到小红书。

> ⚠️ **权威实现**：本 Skill 为 redbook-auto 工具链的 Vault 入口说明，完整功能与最新参数请查阅 [[skills/redbook-auto/README.md|redbook-auto README]] 与 [[skills/redbook-auto/SKILL.md|redbook-auto SKILL]]。

---

## 使用场景

- Creator 直接要求"写一篇小红书笔记"
- Content Agent 调用 redbook-writing 完成 draft 后，进入渲染与发布阶段
- 需要快速生成可发布的小红书图文内容包
- 需要按觉察/重构/创造三层内容适配视觉主题

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| 选题/主题 | string | 内容核心主题 |
| content_layer | string | 觉察层 / 重构层 / 创造层 |
| primary_job | string | feed_stop / search_answer / explain / ... |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| 关键词 | string[] | SEO 标签 |
| 封面文案 | string | cover_title + cover_subtitle |
| 参考案例 | string[] | 对标笔记链接 |

## 工作流程

### 第一步：撰写笔记内容

根据选题和素材，创作符合小红书风格的内容：

**标题**：不超过20字，可用数字/疑问句/感叹号增强吸引力。

**正文**：段落清晰，每段点缀1-2个Emoji，短句短段，结尾附5-10个SEO标签。

**语气规范**（半醒之间版）：
- 探索者语气：不是"我教你"→"我在做，你看"
- 先解决痛苦，再升维：从"明天还要上班的人"的视角出发
- 三层语气适配：
  - 觉察层 → 温柔共鸣
  - 重构层 → 理性启发
  - 创造层 → 真实记录

---

### 第二步：生成渲染用 Markdown

**注意：此Markdown专为图片渲染设计，与正文格式不同。**

```markdown
---
emoji: "🌓"         # 封面装饰Emoji
title: "封面大标题"   # ≤15字
subtitle: "封面副标题" # ≤15字
---

# 正文内容...

---

# 第二张卡片...（手动分页时用 --- 分隔）
```

**分页策略**：
- 内容需精确切分 → 用 `---` 手动分隔，配合 `-m separator`
- 内容长短不稳定 → 生成普通Markdown，使用 `-m auto-split`

---

### 第三步：渲染图片卡片

```bash
python3 skills/redbook-auto/scripts/render_xhs.py <markdown_file> [options]
```

**默认主题**：`sketch`（手绘素描风格）
**默认分页**：`auto-split`（按内容自动切分）

**主题选择**（按内容层推荐）：

| 内容层 | 推荐主题 | 说明 |
|-------|---------|------|
| 觉察层 | `sketch` | 手绘素描，温暖柔和 |
| 重构层 | `default` | 简约干净，聚焦认知 |
| 创造层 | `botanical` | 自然生长，有力量感 |

**分页模式**（`-m`）：

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| `separator` | 按 `---` 分隔符分页 | 内容已手动控量 |
| `auto-split` | 根据高度自动切分 | **推荐，内容长短不稳定** |
| `auto-fit` | 固定尺寸整体缩放 | 封面+单张图 |
| `dynamic` | 动态调整图片高度 | 字数≤550 |

**常用示例**：

```bash
# 觉察层（sketch + 自动分页）
python3 skills/redbook-auto/scripts/render_xhs.py content.md -t sketch -m auto-split

# 重构层（default + 自动分页）
python3 skills/redbook-auto/scripts/render_xhs.py content.md -t default -m auto-split

# 创造层（botanical + 自动分页）
python3 skills/redbook-auto/scripts/render_xhs.py content.md -t botanical -m auto-split

# 指定输出目录
python3 skills/redbook-auto/scripts/render_xhs.py content.md -t sketch -m auto-split -o output/
```

**可用主题**（`-t`）：
`sketch` · `default` · `playful-geometric` · `neo-brutalism` · `botanical` · `professional` · `retro` · `terminal`

**生成结果**：`cover.png`（封面）+ `card_1.png`、`card_2.png`...（正文卡片）

---

### 第四步：发布小红书笔记（可选）

**前置条件**：`skills/redbook-auto/.env` 中配置了 `XHS_COOKIE`

```bash
# 默认仅自己可见（推荐先预览确认）
python3 skills/redbook-auto/scripts/publish_xhs.py \
  --title "笔记标题" \
  --desc "笔记描述" \
  --images cover.png card_1.png card_2.png

# 确认无误后公开发布
python3 skills/redbook-auto/scripts/publish_xhs.py \
  --title "笔记标题" \
  --desc "笔记描述" \
  --images cover.png card_1.png card_2.png --public
```

**推荐流程**：
1. 先用默认（仅自己可见）发布，在App预览确认
2. 确认后删除草稿，用 `--public` 重新发布
3. 或在App中手动转为公开

## 发布前检查清单

- [ ] 标题 ≤20字
- [ ] cover_title ≤15字
- [ ] cover_subtitle ≤15字
- [ ] 正文不含Markdown表格/引用语法（小红书不支持）
- [ ] 正文 ≤1000字
- [ ] 敏感词已过检
- [ ] 图片已渲染完成
- [ ] 没有伪造任何内容
