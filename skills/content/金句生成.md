# 金句生成 Skill

## 1. Skill名称
金句生成

## 2. 使用场景
- 内容中的点睛之笔
- 封面文案设计
- 结尾升华
- 个人IP语录积累

## 3. 输入格式
```yaml
core_message: ""
context: ""
style: "warm" | "sharp" | "philosophical" | "practical"
length: "short" | "medium"
emotion: ""
reference_keywords: []
```

## 4. 执行流程
1. **核心思想提炼**：确定要传递的本质观点
2. **修辞选择**：匹配适合的修辞手法
3. **多版本生成**：同一思想的不同表达
4. **节奏检查**：朗读是否顺口
5. **记忆点评估**：是否容易记住和传播
6. **真实性验证**：是否符合个人观点
7. **语境适配**：是否适合当前内容氛围

### 金句公式库
- **对比型**："{A}不是{B}，{C}才是{D}"
  - 例："自律不是目的，自由才是目的"
- **反转型**："{常识}，但{真相}"
  - 例："努力很重要，但正确的方向更重要"
- **类比型**："{A}就像{B}"
  - 例："成长就像健身，痛过才会变强"
- **排比型**："越{A}，越{B}；越{C}，越{D}"
  - 例："越自律，越自由；越沉淀，越有力量"
- **定义型**："{概念}的本质是{洞察}"
  - 例："成长的本质，是学会与不确定性共处"

## 5. 输出格式
```markdown
## 金句方案
### 首选
{金句}

### 备选
1. {金句}
2. {金句}

### 修辞分析
- 类型：{对比/反转/类比/排比/定义}
- 记忆点：{}
- 推荐位置：{开头/中间/结尾}
```

## 6. 示例
### 输入
message: "行动比完美更重要"
style: "warm"

### 输出
```
## 首选
"完成比完美更重要，先做再做好"

## 备选
1. "你不需要很厉害才开始，你需要开始才厉害"
2. "完美是行动的敌人，不是朋友"

## 修辞分析
- 类型：反转+对比
- 记忆点：完成vs完美的对比
- 推荐位置：结尾升华
```
