# 金句生成

## 使用场景

- 内容中的点睛之笔
- 封面文案设计
- 结尾升华
- 个人 IP 语录积累

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| core_message | string | 要传递的本质观点 |
| style | string | warm / sharp / philosophical / practical |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| context | string | 使用场景 |
| length | string | short / medium |
| emotion | string | 情绪基调 |
| reference_keywords | string[] | 希望包含的关键词 |

## 执行流程

1. **核心思想提炼**：确定要传递的本质观点
2. **修辞选择**：匹配适合的修辞手法
3. **多版本生成**：同一思想的不同表达
4. **节奏检查**：朗读是否顺口
5. **记忆点评估**：是否容易记住和传播
6. **真实性验证**：是否符合个人观点
7. **语境适配**：是否适合当前内容氛围

### 金句公式库
- **对比型**："{A}不是{B}，{C}才是{D}"
  - 例："自律不是目的，自由才是目的"
- **反转型**："{常识}，但{真相}"
  - 例："努力很重要，但正确的方向更重要"
- **类比型**："{A}就像{B}"
  - 例："成长就像健身，痛过才会变强"
- **排比型**："越{A}，越{B}；越{C}，越{D}"
  - 例："越自律，越自由；越沉淀，越有力量"
- **定义型**："{概念}的本质是{洞察}"
  - 例："成长的本质，是学会与不确定性共处"

### 判断标准（if-then）
- 如果金句超过 25 字，则必须压缩或拆分。
- 如果朗读 3 次仍不顺口，则必须重写。
- 如果金句与个人观点不一致，则禁止输出。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 首选金句 | string | — |
| 备选金句 | string[] | 2-3 个 |
| 修辞类型 | string | 对比/反转/类比/排比/定义 |
| 记忆点 | string | 为什么容易记住 |
| 推荐位置 | string | 开头/中间/结尾 |

### 输出示例
```markdown
## 金句方案
### 首选
{金句}

### 备选
1. {金句}
2. {金句}

### 修辞分析
- 类型：{对比/反转/类比/排比/定义}
- 记忆点：{}
- 推荐位置：{开头/中间/结尾}
```

## 示例

### 示例 1：温暖风格
**输入**：
```yaml
core_message: "行动比完美更重要"
style: "warm"
```

**输出**：
```markdown
## 金句方案
### 首选
"完成比完美更重要，先做再做好"

### 备选
1. "你不需要很厉害才开始，你需要开始才厉害"
2. "完美是行动的敌人，不是朋友"

## 修辞分析
- 类型：反转+对比
- 记忆点：完成 vs 完美的对比
- 推荐位置：结尾升华
```
