# 互动运营 Skill

## 1. Skill名称
互动运营

## 2. 使用场景
- 评论区管理运营
- 粉丝关系维护
- 私域引流转化
- 提高内容互动率

## 3. 输入格式
```yaml
content_posted:
  title: ""
  url: ""
  publish_time: ""
comments_received: []
current_interaction_rate: 0
goal: "increase_engagement" | "lead_to_private" | "build_community"
```

## 4. 执行流程
1. **评论分类处理**：问题型/共鸣型/反馈型/争议型
2. **问题型回复**：提供延伸价值
3. **共鸣型互动**：强化情感连接
4. **引导话术设计**：自然引导到私域/下篇内容
5. **互动节奏控制**：发布后1小时内密集互动
6. **高价值评论挖掘**：识别能激发二次传播的评论
7. **评论区置顶策略**：用置顶评论引导互动方向

### 评论回复策略
- **问题类**：认真回答+引导看相关笔记
- **共鸣类**：感谢+延伸共鸣+提问互动
- **反馈类**：感谢建议+讨论可行性
- **争议类**：理性讨论+求同存异
- **广告类**：直接删除

### 引导话术模板
- "关于这个问题，我之前在{xx}笔记里详细讲过，可以去看看~"
- "好多姐妹问这个，要不要我专门出一期？"
- "你觉得呢？欢迎在评论区分享你的方法~"
- "如果有帮助，记得点个收藏方便之后找~"

## 5. 输出格式
```markdown
## 互动运营方案
### 内容信息
- 标题：{}
- 发布时间：{}

### 评论回复模板
1. {评论类型}：{回复模板}
2. {评论类型}：{回复模板}

### 置顶评论建议
{置顶评论内容}

### 互动目标
- 目标互动率：{}
- 重点互动时段：{}
```

## 6. 示例
```markdown
## 互动运营方案
### 内容信息
- 标题："建立知识体系后，我的效率提升了3倍"
- 发布时间：今晚20:00

### 评论回复模板
1. 问题类："关于知识管理的工具，我用的是Obsidian，具体方法在我置顶笔记里有详细教程哦~"
2. 共鸣类："太懂你了！我之前也是这种感觉，慢慢来，你可以的 💪"

### 置顶评论建议
"大家平时用什么工具管理知识？评论区分享一下吧~"

### 互动目标
- 目标互动率：5%+
- 重点互动时段：发布后2小时
```
