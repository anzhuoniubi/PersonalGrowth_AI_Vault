# 互动运营

## 使用场景

- 评论区管理运营
- 粉丝关系维护
- 私域引流转化
- 提高内容互动率

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| content_posted.title | string | 已发布内容标题 |
| content_posted.url | string | 内容链接 |
| content_posted.publish_time | string | 发布时间 |
| goal | string | increase_engagement / lead_to_private / build_community |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| comments_received | object[] | 已收到的评论列表 |
| current_interaction_rate | number | 当前互动率 |

## 执行流程

1. **评论分类处理**：将评论分为问题型 / 共鸣型 / 反馈型 / 争议型 / 广告型
2. **问题型回复**：提供延伸价值，引导看相关笔记
3. **共鸣型互动**：强化情感连接，适度提问
4. **引导话术设计**：自然引导到私域/下篇内容
5. **互动节奏控制**：发布后 1 小时内密集互动
6. **高价值评论挖掘**：识别能激发二次传播的评论
7. **评论区置顶策略**：用置顶评论引导互动方向

### 评论回复策略
- **问题类**：认真回答 + 引导看相关笔记
- **共鸣类**：感谢 + 延伸共鸣 + 提问互动
- **反馈类**：感谢建议 + 讨论可行性
- **争议类**：理性讨论 + 求同存异
- **广告类**：直接删除

### 判断标准（if-then）
- 如果评论含具体问题，则必须在 30 分钟内回复。
- 如果评论表达强烈共鸣，则回复中必须包含一个开放式问题。
- 如果评论含争议且可能升级，则先私信沟通，不在评论区争辩。
- 如果评论是广告/引流，则直接删除并记录。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 内容信息 | object | 标题/链接/发布时间 |
| 评论回复模板 | object[] | 按评论类型分类的回复模板 |
| 置顶评论建议 | string | 用于置顶的引导评论 |
| 互动目标 | object | 目标互动率/重点时段 |

### 输出示例
```markdown
## 互动运营方案
### 内容信息
- 标题：{}
- 发布时间：{}

### 评论回复模板
1. 问题类：{回复模板}
2. 共鸣类：{回复模板}
3. 反馈类：{回复模板}
4. 争议类：{回复模板}

### 置顶评论建议
{置顶评论内容}

### 互动目标
- 目标互动率：{}
- 重点互动时段：{}
```

## 示例

### 示例 1：觉察层内容评论区运营
**输入**：
```yaml
content_posted:
  title: "20多岁最怕的不是没钱，是在过二手人生"
  url: "https://..."
  publish_time: "2026-07-22 20:00"
goal: "increase_engagement"
```

**输出**：
```markdown
## 互动运营方案
### 内容信息
- 标题：20多岁最怕的不是没钱，是在过二手人生
- 发布时间：2026-07-22 20:00

### 评论回复模板
1. 问题类："你提到的'二手人生'具体指什么？" → "简单说就是按别人的剧本生活。关于怎么找到自己的人生主线，我下周会专门写一篇，到时候@你~"
2. 共鸣类："真的是我，每天都在完成任务但不知道为了什么" → "这种感觉很真实。你现在最想做但还没开始的一件事是什么？"
3. 反馈类："写得很好，但例子可以再具体一点" → "谢谢你的建议，我会记下来。能不能告诉我你期待看到什么样的例子？"
4. 争议类："20多岁没钱也很可怕好吧" → "你说得对，经济压力确实真实存在。我的意思是，除了没钱，'不知道自己想要什么'可能是更隐蔽的焦虑源。你怎么看？"

### 置顶评论建议
"你也有过'在过二手人生'的感觉吗？是在哪个瞬间意识到的？"

### 互动目标
- 目标互动率：5%+
- 重点互动时段：发布后 2 小时
```

### 示例 2：方法论类内容私域引导
**输入**：
```yaml
content_posted:
  title: "3个习惯帮我走出内耗"
  url: "https://..."
  publish_time: "2026-07-22 20:00"
goal: "lead_to_private"
```

**输出**：
```markdown
## 互动运营方案
### 置顶评论建议
"很多姐妹问具体怎么开始，我把这 3 个习惯的执行清单整理成了一张图。需要的扣 1，我私信发你~"

### 高频回复模板
- "蹲" → "发你啦，记得看私信~"
- "怎么领" → "私信回复'内耗'就能收到自动回复~"
- "真的有用吗" → "我坚持了 90 天，最大的变化是晚上不再反复回想白天的事。你可以先试 7 天看看~"
```
