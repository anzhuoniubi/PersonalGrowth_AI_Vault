# 增长策略

## 使用场景

- 账号冷启动增长
- 粉丝增长瓶颈突破
- 跨平台引流策略
- 内容分发优化

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| current_status.followers | number | 当前粉丝数 |
| current_status.avg_views | number | 平均阅读 |
| current_status.engagement_rate | number | 互动率 |
| growth_goal.target_followers | number | 目标粉丝数 |
| growth_goal.timeline | string | 时间周期 |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| constraints.daily_time | string | 每日可投入时间 |
| constraints.content_type | string | 内容类型 |
| platform | string | 默认 xiaohongshu |

## 执行流程

1. **现状诊断**：分析当前增长瓶颈
2. **对标分析**：研究同期增长快的账号
3. **杠杆点识别**：什么内容带来最多增长
4. **增长渠道评估**：自然流量 vs 搜索流量 vs 推荐流量
5. **策略制定**：确定增长杠杆组合
6. **执行计划**：制定周度增长行动项
7. **效果追踪**：建立增长指标看板

### 增长杠杆模型
- **内容杠杆**：爆款内容带来的指数增长
- **搜索杠杆**：长尾关键词持续流量
- **互动杠杆**：评论互动提升推荐权重
- **合作杠杆**：互推、合集、联动
- **跨平台杠杆**：各平台互相引流

### 判断标准（if-then）
- 如果周增长低于目标 50%，则优先启动内容杠杆。
- 如果搜索流量占比 <30%，则优先启动搜索杠杆。
- 如果互动率 <5%，则优先启动互动杠杆。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 现状诊断 | object | 增长速率/流量来源/瓶颈 |
| 增长杠杆 | object[] | 杠杆+预期效果 |
| 30 天行动计划 | object | 每周任务 |
| 关键指标 | object | 周增长目标/核心行动项 |

## 示例

### 示例 1：冷启动增长
**输入**：
```yaml
current_status:
  followers: 500
  avg_views: 300
  engagement_rate: 0.06
growth_goal:
  target_followers: 2000
  timeline: "30天"
constraints:
  daily_time: "2小时"
  content_type: "图文"
platform: "xiaohongshu"
```

**输出**：
```markdown
## 增长策略报告
### 现状诊断
- 当前增长速率：50 粉/周
- 主要流量来源：搜索流量占 60%
- 增长瓶颈：缺乏爆款内容，互动率偏低

### 增长杠杆
1. 内容杠杆 - 预期效果：打造 1 篇万赞爆款
2. 搜索杠杆 - 预期效果：覆盖 20 个长尾关键词
3. 互动杠杆 - 预期效果：评论率从 1% 提升至 3%

### 30天行动计划
第1周：分析竞品爆款，确定 5 个高潜力选题
第2周：产出 2 篇高潜力内容，测试不同标题公式
第3周：互动策略，发布后 1 小时内密集回复评论
第4周：复盘优化，放大有效策略

### 关键指标
- 周增长目标：375 粉/周
- 核心行动项：每周发布 3 篇，其中 1 篇为搜索型长尾内容
```
