# 增长策略 Skill

## 1. Skill名称
增长策略

## 2. 使用场景
- 账号冷启动增长
- 粉丝增长瓶颈突破
- 跨平台引流策略
- 内容分发优化

## 3. 输入格式
```yaml
current_status:
  followers: 0
  avg_views: 0
  engagement_rate: 0
  content_frequency: ""
growth_goal:
  target_followers: 0
  timeline: ""
constraints:
  daily_time: ""
  content_type: ""
platform: "xiaohongshu"
```

## 4. 执行流程
1. **现状诊断**：分析当前增长瓶颈
2. **对标分析**：研究同期增长快的账号
3. **杠杆点识别**：什么内容带来最多增长
4. **增长渠道评估**：自然流量 vs 搜索流量 vs 推荐流量
5. **策略制定**：确定增长杠杆组合
6. **执行计划**：制定周度增长行动项
7. **效果追踪**：建立增长指标看板

### 增长杠杆模型
- **内容杠杆**：爆款内容带来的指数增长
- **搜索杠杆**：长尾关键词持续流量
- **互动杠杆**：评论互动提升推荐权重
- **合作杠杆**：互推、合集、联动
- **跨平台杠杆**：各平台互相引流

## 5. 输出格式
```markdown
## 增长策略报告
### 现状诊断
- 当前增长速率：{周均增长}
- 主要流量来源：{}
- 增长瓶颈：{}

### 增长杠杆
1. {杠杆1} - 预期效果：{}
2. {杠杆2} - 预期效果：{}

### 30天行动计划
第1周：{}
第2周：{}
第3周：{}
第4周：{}

### 关键指标
- 周增长目标：{}
- 核心行动项：{}
```

## 6. 示例
```markdown
## 增长策略报告
### 现状诊断
- 当前增长速率：50粉/周
- 主要流量来源：搜索流量占60%
- 增长瓶颈：缺乏爆款内容

### 增长杠杆
1. 内容杠杆 - 预期效果：打造1篇万赞爆款
2. 搜索杠杆 - 预期效果：覆盖20个长尾关键词

### 30天行动计划
第1周：分析竞品爆款，确定选题方向
第2周：产出2篇高潜力内容
第3周：互动策略，提升评论率
第4周：复盘优化，放大有效策略
```
