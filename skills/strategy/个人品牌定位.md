# 个人品牌定位 Skill

## 1. Skill名称
个人品牌定位（半醒之间版）

## 2. 使用场景
- 账号冷启动阶段的定位设计
- 每季度品牌定位回顾与调整 → 对齐 [[01_个人定位/半醒之间IP定位]]
- 内容方向跑偏时的纠偏决策
- **三层内容体系校验**：当前选题/内容是否在"觉察-重构-创造"的框架内

## 3. 输入格式
```yaml
creator_info:
  name: ""
  background: ""
  expertise: []
  personality_traits: []
  values: []
market_info:
  target_audience: ""
  competitors: []
  market_trends: []
goals:
  short_term: ""
  long_term: ""
```

## 4. 执行流程
1. **自我分析**：梳理个人优势、经历、独特视角
2. **市场扫描**：分析竞品定位、用户需求空白
3. **定位假设**：提出3个可能的定位方向
4. **差异化验证**：评估每个方向的独特性
5. **情感共鸣测试**：目标用户是否会为此停留
6. **可持续性评估**：能否持续产出相关内容
7. **定位确定**：输出最终定位声明

## 5. 输出格式
```markdown
## 定位声明
{一句话说明我是谁、为谁、提供什么价值}

## 差异化优势
1. {独特优势1}
2. {独特优势2}

## 目标用户
{精准用户画像}

## 内容边界
- 做：{内容范围}
- 不做：{内容红线}

## 表达基调
{语气、风格、态度描述}
```

## 6. 示例
### 输入
```yaml
creator_info:
  name: "小明"
  background: "5年互联网产品经理"
  expertise: ["用户研究", "数据分析", "产品思维"]
  personality_traits: ["理性", "温和", "有条理"]
  values: ["真实", "长期主义", "利他"]
```

### 输出
```
## 定位声明
我是小明，一个用产品经理思维做个人成长的同行者，
帮25-35岁职场人用系统方法实现可持续成长。

## 差异化优势
1. 产品经理的理性视角 × 个人成长的感性需求
2. 强调"可执行的系统"而非碎片化技巧
3. 用数据验证方法有效性，不空谈理论
```
