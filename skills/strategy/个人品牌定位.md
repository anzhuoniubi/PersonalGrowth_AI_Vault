# 个人品牌定位

## 使用场景

- 账号冷启动阶段的定位设计
- 每季度品牌定位回顾与调整 → 对齐 [[01_个人定位/半醒之间IP定位.md|半醒之间IP定位]]
- 内容方向跑偏时的纠偏决策
- 三层内容体系校验：当前选题/内容是否在"觉察-重构-创造"框架内

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| creator_info.name | string | 创作者名称 |
| creator_info.background | string | 背景经历 |
| creator_info.expertise | string[] | 专业能力 |
| creator_info.personality_traits | string[] | 人格特质 |
| creator_info.values | string[] | 价值观 |
| market_info.target_audience | string | 目标用户 |
| goals.short_term | string | 短期目标 |
| goals.long_term | string | 长期目标 |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| market_info.competitors | string[] | 竞品列表 |
| market_info.market_trends | string[] | 市场趋势 |

## 执行流程

1. **自我分析**：梳理个人优势、经历、独特视角
2. **市场扫描**：分析竞品定位、用户需求空白
3. **定位假设**：提出 3 个可能的定位方向
4. **差异化验证**：评估每个方向的独特性
5. **情感共鸣测试**：目标用户是否会为此停留
6. **可持续性评估**：能否持续产出相关内容
7. **定位确定**：输出最终定位声明

### 判断标准（if-then）
- 如果定位方向与 [[01_个人定位/半醒之间IP定位.md|半醒之间IP定位]] 冲突，则必须排除。
- 如果无法在 6 个月内持续产出内容，则必须调整定位。
- 如果目标用户无法 3 秒理解定位，则必须简化表达。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 定位声明 | string | 一句话说明我是谁、为谁、提供什么价值 |
| 差异化优势 | string[] | 独特优势 |
| 目标用户 | string | 精准画像 |
| 内容边界 | object | 做/不做 |
| 表达基调 | string | 语气/风格/态度 |

### 输出示例
```markdown
## 定位声明
{一句话说明我是谁、为谁、提供什么价值}

## 差异化优势
1. {独特优势1}
2. {独特优势2}

## 目标用户
{精准用户画像}

## 内容边界
- 做：{内容范围}
- 不做：{内容红线}

## 表达基调
{语气、风格、态度描述}
```

## 示例

### 示例 1：产品经理个人成长定位
**输入**：
```yaml
creator_info:
  name: "小明"
  background: "5年互联网产品经理"
  expertise: ["用户研究", "数据分析", "产品思维"]
  personality_traits: ["理性", "温和", "有条理"]
  values: ["真实", "长期主义", "利他"]
market_info:
  target_audience: "25-35岁职场人"
  competitors: ["A", "B"]
goals:
  short_term: "1000粉"
  long_term: "成为个人成长领域可信的声音"
```

**输出**：
```markdown
## 定位声明
我是小明，一个用产品经理思维做个人成长的同行者，
帮 25-35 岁职场人用系统方法实现可持续成长。

## 差异化优势
1. 产品经理的理性视角 × 个人成长的感性需求
2. 强调"可执行的系统"而非碎片化技巧
3. 用数据验证方法有效性，不空谈理论

## 目标用户
25-35 岁职场人，工作 3-8 年，感到成长停滞，希望找到系统方法突破瓶颈。

## 内容边界
- 做：个人成长方法论、认知升级、真实实践经验
- 不做：成功学、贩卖焦虑、无法验证的鸡汤

## 表达基调
理性但不冰冷，温和但有边界，像一个有经验的同行者。
```
