# 赛道分析

## 使用场景

- 月度战略规划时，研判 IP 所在赛道的趋势与竞争格局
- 进入新内容类目或调整定位前，评估机会与风险
- CEO Agent 季度品牌定位评估的输入之一

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| 赛道名称 | string | 如「个人成长」「AI 时代个体价值」 |
| 分析时间窗口 | string | 如「近 30 天 / 近 90 天」 |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| 竞品账号列表 | string[] | 3-5 个对标账号主页链接 |
| 关键词池 | string[] | 用于搜索采样的核心关键词 |

## 执行流程

1. **定义赛道边界**
   - 明确赛道核心主题、目标用户、内容形态
   - 列出赛道内 3-5 个核心关键词

2. **数据采集**
   - 搜索核心关键词，记录头部笔记的阅读、互动、发布时间
   - 收集竞品账号近期 10 篇内容数据
   - 记录平台算法信号：推荐页出现频率、话题热度

3. **竞争格局分析**
   - 头部账号：内容定位、更新频率、爆款率
   - 腰部账号：差异化角度、增长趋势
   - 空白机会：未被充分满足的用户需求

4. **趋势判断**
   - 近期热度变化：上升 / 平稳 / 下降
   - 新兴话题：近 30 天内出现的新角度
   - 平台机制变化：对内容形态/发布频率的影响

5. **输出赛道研判报告**

### 判断标准（if-then）
- 如果头部账号爆款率 ≥30%，则赛道进入红海，需寻找细分差异点。
- 如果近 30 天新兴话题占比 ≥20%，则赛道处于变化期，有机会窗口。
- 如果目标用户需求与现有头部供给存在明显错配，则标记为「可切入机会」。
- 如果数据不足，则明确标注「数据待补充」，不强行下结论。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 赛道名称 | string | 本次分析的赛道 |
| 热度判断 | string | 上升/平稳/下降 |
| 竞争强度 | string | 红海/黄海/蓝海 |
| 机会点 | string[] | 可切入的细分机会 |
| 风险提示 | string[] | 主要风险 |
| 行动建议 | string | 对选题/定位的具体建议 |
| 数据置信度 | string | 高/中/低 |

### 输出示例
```markdown
== 赛道研判报告 ==
赛道名称：AI 时代个人价值重构
分析窗口：2026-06-22 至 2026-07-22
热度判断：上升
竞争强度：黄海（头部已出现，但细分机会仍在）

== 头部账号特征 ==
1. {账号名} · {定位} · {爆款率}% · 数据待补充

== 机会点 ==
1. {未被满足的需求} · 证据：{链接或观察}

== 风险提示 ==
1. {风险描述}

== 行动建议 ==
{具体建议}

== 数据置信度 ==
中（样本量待补充）
```

## 示例

### 示例 1：月度赛道扫描
**输入**：
```markdown
赛道名称：AI 时代个人价值重构
分析时间窗口：近 30 天
关键词池：["AI 替代", "一人公司", "个人品牌", "AI 工作流"]
```

**输出**：
```markdown
== 赛道研判报告 ==
赛道名称：AI 时代个人价值重构
分析窗口：近 30 天
热度判断：上升
竞争强度：黄海
机会点：
1. 「AI 时代普通人的认知升级路径」角度供给不足
2. 「从 0 到 1 搭建个人 AI 工作流」的实操记录稀缺
风险提示：
1. 头部账号开始布局，窗口期约 60-90 天
2. 平台对 AI 工具推荐类内容审核趋严
行动建议：本月优先生产 2-3 篇「觉察层」内容建立共鸣，再切入重构层方法论
数据置信度：中
```
