# 封面设计

## 使用场景

- 每篇小红书笔记封面设计 → 对齐「半醒之间」视觉语言
- 系列内容封面统一风格 → 体现"电影感、光与暗、半醒、温暖不甜腻"
- 提高笔记点击率

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| content_core | string | 内容核心表达点 |
| content_layer | string | 觉察层 / 重构层 / 创造层 |
| content_type | string | 情绪共鸣 / 认知观点 / 干货方法论 / 个人叙事 / 对比 / 系列 |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| title | string | 主标题 |
| subtitle | string | 副标题 |
| keywords | string[] | 希望突出的关键词 |

## 执行流程

1. **内容分析**：从正文提取最核心的一个表达点——只能选一个
2. **选择构图**：根据内容类型从六种构图里选一种
3. **填写三区**：上 1/3 主标题 / 中 1/2 视觉区 / 下 1/3 辅助区
4. **配色+字体确认**：主色/辅色/点缀色 + 字体
5. **小图测试**：把封面缩到手机屏幕大小检查可读性
6. **输出方案**

### 品牌视觉对齐（流程内小节）
**视觉人格关键词**：`电影感` `光与暗` `半醒状态` `觉察氛围` `温暖但不甜腻`

**三层内容 × 封面情绪**：
| 内容层 | 封面情绪 | 配色倾向 | 视觉隐喻 |
|-------|---------|---------|---------|
| 觉察层 | 温柔·共鸣·疗愈 | 暖棕/焦糖+米白+深灰 | 光穿过缝隙、半明半暗 |
| 重构层 | 理性·认知·启发 | 深蓝/墨蓝+白+暖金 | 新的地图、打开的窗 |
| 创造层 | 真实·行动·力量 | 墨绿/苔绿+米白+暖金 | 生长的植物、建筑的过程 |

### 核心框架：三阶式排版
```
┌─────────────────┐
│                  │
│   上 1/3         │  ← 主标题区
│   📌 主标题       │     字体粗壮，6-12字以内
│   此处是核心      │     最具冲击力的信息放在这里
│                  │
├─────────────────┤
│                  │
│                  │
│   中 1/2         │  ← 视觉主体区
│                  │     图片/人物/场景
│                  │     清晰锐利，层次分明
│                  │
│                  │
├─────────────────┤
│   下 1/3         │  ← 辅助信息区
│   副标题/标签      │     小字，补充信息
│                  │     不是必须的，宁少勿多
└─────────────────┘
```

| 区域 | 占画面 | 放什么 | 字体 | 字号 |
|------|--------|--------|------|------|
| **上1/3 主标题区** | 30% | 核心标题 / 人群标签 + 情绪钩子 | 思源黑体 Bold | 最大 |
| **中1/2 视觉区** | 50% | 人物 / 场景 / 对比图 / 纯色底 | 不叠文字 | — |
| **下1/3 辅助区** | 20% | 副标题 / 引导语 / 金句（可选） | 思源黑体 Regular | 小 |

### 字体规则（不可妥协）
- 只用 **1 种粗体**（思源黑体 Bold / 阿里巴巴普惠体 Bold / 优设标题黑）
- 只用 **1 种颜色**：白字或黑字，最多加 1 种点缀色
- 禁用：宋体、楷体、手写体、花字、渐变字、旋转字、超过 3 种颜色

### 六种构图公式
| 内容类型 | 推荐构图 | 上1/3 | 中1/2 | 下1/3 |
|---------|---------|-------|-------|-------|
| 情绪共鸣/觉察层 | 中心聚焦 | 人群标签+情绪钩 | 人物/剪影居中 | 金句小字（可选） |
| 认知观点/反转类 | 大字报 | 主标题占满上+中 | （可选）极简配图 | 副标题 |
| 干货方法论类 | 多图拼贴 | 数字+结果 | 3个图示拼贴 | 标签/关键词 |
| 个人叙事类 | 场景代入 | 情绪钩子 | 真实场景/环境 | 引导语 |
| 对比类 | 对比构图 | "以前vs现在" | 左右/上下对比图 | 结果标签 |
| 系列内容 | 固定模板 | 统一格式，只换文字 | 同上 | 同上 |

### 判断标准（if-then）
- 如果主标题 >12 字，则必须压缩。
- 如果小图模式下主标题无法 3 秒识别，则必须重写或放大。
- 如果配色 >3 种，则必须删减。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 选中构图 | string | 中心聚焦/大字报/多图拼贴/场景代入/对比/固定模板 |
| 三区布局 | object | 上/中/下内容 |
| 配色 | object | 背景色/文字色/点缀色 |
| 制作方式 | string | AI 出图 / 手机拍摄 / 纯文字排版 |
| 避坑检查 | object | 检查项结果 |

### 输出示例
```markdown
## 封面方案
**选中构图**：{}

| 区域 | 内容 | 颜色/字体 |
|------|------|-----------|
| 上1/3 | {主标题} | 白/黑，思源黑体Bold |
| 中1/2 | {画面描述} | — |
| 下1/3 | {可选} | 小字，同字体 |

### 配色
- 背景色：{}
- 文字色：{}
- 点缀色：{}（可选）

### 制作方式
{AI出图 / 手机拍摄 / 纯文字排版}
```

## 示例

### 示例 1：觉察层情绪共鸣封面
**输入**：
```yaml
content_core: "二十多岁在过二手人生"
content_layer: "觉察层"
content_type: "情绪共鸣"
title: "二手人生"
subtitle: "你也在过吗"
```

**输出**：
```markdown
## 封面方案
**选中构图**：中心聚焦

### 三区布局
| 区域 | 内容 | 颜色/字体 |
|------|------|-----------|
| 上1/3 | 二手人生 | 米白色，思源黑体Bold，60pt |
| 中1/2 | 窗边独坐的人影，晨光从窗户斜射进来 | — |
| 下1/3 | 你也在过吗 | 暖金色，思源黑体Regular，24pt |

### 配色
- 背景色：暖棕 #B76E4A 渐变至深棕
- 文字色：米白 #F5F0E8
- 点缀色：暖金 #C9A96E

### 制作方式
AI 出图（调用 [[skills/visual/AI绘图提示词.md|AI绘图提示词]]）+ Canva 叠加文字

### 避坑检查
- [x] 标题不超过 12 字
- [x] 字色与背景高对比
- [x] 全图 ≤3 种颜色
- [x] 只用 1 种字体
- [x] 小图模式下可读
- [x] 没有违禁词
- [x] 封面和正文内容一致
```
