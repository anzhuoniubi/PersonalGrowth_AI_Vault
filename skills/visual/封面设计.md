## 1. Skill名称
封面设计（半醒之间版）

## 2. 使用场景
- 每篇小红书笔记封面设计 → 对齐「半醒之间」视觉语言
- 系列内容封面统一风格 → 体现"电影感、光与暗、半醒、温暖不甜腻"
- 提高笔记点击率

## 2.5 品牌视觉对齐（半醒之间）
每次设计前，确认以下品牌要素：

### 视觉人格关键词
`电影感` `光与暗` `半醒状态` `觉察氛围` `温暖但不甜腻`

### 三层内容 × 封面情绪
| 内容层 | 封面情绪 | 配色倾向 | 视觉隐喻 |
|-------|---------|---------|---------|
| 觉察层 | 温柔·共鸣·疗愈 | 暖棕/焦糖+米白+深灰 | 光穿过缝隙、半明半暗 |
| 重构层 | 理性·认知·启发 | 深蓝/墨蓝+白+暖金 | 新的地图、打开的窗 |
| 创造层 | 真实·行动·力量 | 墨绿/苔绿+米白+暖金 | 生长的植物、建筑的过程 |

## 3. 核心框架：三阶式排版

> 市场数据：传统居中堆砌式CTR仅3.2%，三阶式CTR达**12.8%**

```
┌─────────────────┐
│                  │
│   上 1/3         │  ← 主标题区
│   📌 主标题       │     字体粗壮，6-12字以内
│   此处是核心      │     最具冲击力的信息放在这里
│                  │
├─────────────────┤
│                  │
│                  │
│   中 1/2         │  ← 视觉主体区
│                  │     图片/人物/场景
│                  │     清晰锐利，层次分明
│                  │
│                  │
├─────────────────┤
│   下 1/3         │  ← 辅助信息区
│   副标题/标签      │     小字，补充信息
│                  │     不是必须的，宁少勿多
└─────────────────┘
```

### 三区的设计规范

| 区域 | 占画面 | 放什么 | 字体 | 字号 |
|------|--------|--------|------|------|
| **上1/3 主标题区** | 30% | 核心标题 / 人群标签 + 情绪钩子 | 思源黑体Bold / 阿里巴巴普惠体Bold | 最大，占满该区 |
| **中1/2 视觉区** | 50% | 人物 / 场景 / 对比图 / 纯色底 | 不叠文字，或仅极小标签 | 不干扰主体 |
| **下1/3 辅助区** | 20% | 副标题 / 引导语 / 金句（可选） | 思源黑体Regular | 小，清晰即可 |

### 字体规则（不可妥协）
- 只用**1种粗体**（推荐：思源黑体Bold / 阿里巴巴普惠体Bold / 优设标题黑）
- 只用**1种颜色**：白字 或 黑字（视背景对比度），最多加1种点缀色
- 禁用：宋体、楷体、手写体、花字、渐变字、旋转字、超过3种颜色

### 配色规则
- 全图不超过**3种颜色**
- 背景色：纯色或简单渐变，不要复杂图片作底
- 文字色：白 或 黑，与背景高对比
- 点缀色（可选）：1种，面积不超过10%

## 4. 六种已验证的构图公式

根据你的内容类型选择一个：

| 内容类型 | 推荐构图 | 上1/3 | 中1/2 | 下1/3 |
|---------|---------|-------|-------|-------|
| **情绪共鸣/觉察层** | 中心聚焦 | 人群标签+情绪钩 | 人物/剪影居中 | 金句小字（可选） |
| **认知观点/反转类** | 大字报 | 主标题占满上+中 | （可选）极简配图 | 副标题 |
| **干货方法论类** | 多图拼贴 | 数字+结果 | 3个图示拼贴 | 标签/关键词 |
| **个人叙事类** | 场景代入 | 情绪钩子 | 真实场景/环境 | 引导语 |
| **对比类** | 对比构图 | "以前vs现在" | 左右/上下对比图 | 结果标签 |
| **系列内容** | 固定模板 | 统一格式，只换文字 | 同上 | 同上 |

## 5. 执行流程

### Step 1：内容分析
从正文提取最核心的一个表达点——只能选一个，封面放不下第二个。

### Step 2：选择构图
根据内容类型从上面六种里选一种。

### Step 3：填写三区
```
上1/3：______（主标题，6-12字）
中1/2：______（视觉画面描述）
下1/3：______（可选，不一定要填）
```

### Step 4：配色+字体确认
- 主色：______
- 辅色：______
- 点缀色：______（不超过10%面积）
- 字体：思源黑体Bold（只用这一种）

### Step 5：小图测试
把封面缩到手机屏幕大小（约3cm宽），检查：
- [ ] 主标题能看清吗？
- [ ] 3秒内能理解在说什么吗？
- [ ] 画面主体不会被文字遮挡吗？
- [ ] 颜色不刺眼也不灰暗吗？

### Step 6：输出方案

## 6. 输出格式

```markdown
## 封面方案
**选中构图**：{中心聚焦 / 大字报 / 多图拼贴 / 场景代入 / 对比 / 固定模板}

### 三区布局
| 区域 | 内容 | 颜色/字体 |
|------|------|-----------|
| 上1/3 | {主标题} | 白/黑，思源黑体Bold |
| 中1/2 | {画面描述} | — |
| 下1/3 | {可选} | 小字，同字体 |

### 配色
- 背景色：{}
- 文字色：{}
- 点缀色：{}（可选）

### 制作方式
{AI出图 / 手机拍摄 / 纯文字排版}
```

## 7. 避坑清单

- [ ] 标题不超过12字
- [ ] 字色与背景高对比
- [ ] 全图≤3种颜色
- [ ] 只用1种字体
- [ ] 小图模式下可读
- [ ] 没有"最""第一""绝了"等违禁词
- [ ] 封面和正文内容一致
- [ ] 没有过度P图失真
