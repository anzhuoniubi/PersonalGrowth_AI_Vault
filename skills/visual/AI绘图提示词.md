## 1. Skill名称
AI绘图提示词（半醒之间版）

## 2. 使用场景
- 封面背景图生成（必须应用全局基底 + 品牌视觉关键词）
- 文章配图设计
- 品牌视觉素材制作

## 2.5 品牌视觉关键词（半醒之间）
写提示词时融入以下品牌视觉语言：

**核心氛围**：电影感、光与暗的对比、半醒状态、觉察氛围、温暖但不甜腻

**三层内容 × 情绪基调参考**：
- 觉察层 → 温暖柔光、半明半暗、静谧内省
- 重构层 → 明亮开阔、新视角、清晰线条
- 创造层 → 生长感、质感、手工感、真实纹理

## 3. 全局封面基底（每次必用）

每次写封面提示词时，**这段内容必须作为固定前缀**，不可省略或修改。
颜色方案不再固定为暗夜灰蓝，而是根据每篇内容的**核心表达点**和**情绪基调**自定义。

```
小红书封面，3：4比例。电影感光影，细腻质感，温暖氛围。

画面留白充分，确保文字可读。主体居中或偏一侧，不占据超过40%的画面空间。
构图干净，没有多余的装饰元素。

安静、孤独但并不绝望。有一种"我选择不加入"的平静感，而不是"我被抛弃了"的沉溺感。
```

### 颜色方案：从内容中提取

不再使用固定色板。每次封面的颜色方案由内容的**核心表达点**决定：

| 内容情绪 | 推荐配色方向 | 示例 |
|---------|------------|------|
| 冷静·认知·理性 | 深蓝/墨蓝 + 白 + 单色点缀 | 系统、框架、逻辑类 |
| 温暖·共鸣·疗愈 | 暖棕/焦糖 + 米白 + 深灰 | 情绪共鸣、内耗类 |
| 锐利·反差·观点 | 深色底 + 高对比亮色点缀 | 认知反转、争议类 |
| 自然·生长·沉淀 | 墨绿/苔绿 + 米白 + 暖金 | 成长、时间积累类 |
| 极简·干净·留白 | 纯色浅底 + 深色文字 | 方法论、清单类 |

配色原则（不可妥协）：
- 全图不超过 **3种颜色**
- 文字色与背景**高对比**（白底黑字 或 深底白字）
- 点缀色只占 **≤10%** 画面
- 不使用高饱和荧光色

### 使用方式（公式）

```
[全局基底]
[当次配色方案：主色 + 文字色 + 点缀色]
[画面主体描述]
[文字叠加上去：具体文案]
```

## 5. 执行流程

1. **前置全局基底**（不可跳过）
2. **概念抽象**：将内容核心转化为视觉画面
3. **元素选择**：控制在1-2个核心元素，不超过3个
4. **文字层级设计**：顶部（小字引题）→ 中央（大字主标题）→ 底部（小字金句/点缀）
5. **合成完整提示词** = 全局基底 + 画面描述 + 文字叠加
6. **发布前检查**：
   - [ ] 全局基底是否完整包含？
   - [ ] 冷暖比例是否90:10？
   - [ ] 文字是否超过40%画面？
   - [ ] 情绪是否为"孤独但有力量"而非绝望？

## 6. 避坑清单

- ❌ 不要使用高饱和色彩
- ❌ 不要让画面过于复杂（元素≤3个）
- ❌ 不要让文字占据超过40%画面
- ❌ 不要使用恐怖、阴森、色情元素
- ❌ 不要让情绪沦落为"可怜""绝望"
