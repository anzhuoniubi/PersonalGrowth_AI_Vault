# 私域运营 Skill

## 1. Skill名称
私域运营

## 2. 使用场景
- 小红书粉丝向私域导流
- 私域社群运营
- 用户转化和复购
- 建立长期用户关系

## 3. 输入格式
```yaml
current_private_domain:
  platform: "wechat" | "微信群"
  existing_users: 0
  daily_new: 0
traffic_source:
  main_channel: "xiaohongshu"
  daily_views: 0
  conversion_rate: 0
goal: "growth" | "activation" | "conversion" | "retention"
resource_available:
  daily_time: ""
  team_size: 1
```

## 4. 执行流程
1. **导流路径设计**：评论区→私信→微信/社群
2. **诱饵设计**：用户为什么愿意加微信
3. **承接流程**：加好友后的自动化流程
4. **信任建立**：持续提供价值建立信任
5. **分层运营**：按活跃度和付费意愿分层
6. **转化时机**：识别最佳转化节点
7. **社群运营**：社群活跃度和价值交付

### 导流方式
- **评论区引导**："整理了完整版PDF，需要的姐妹扣1"
- **私信自动回复**：关键词触发资料包
- **主页简介**：微信号/社群入口
- **直播引流**：直播中引导加群

### 私域内容策略
- **朋友圈**：日常分享+干货片段+生活感悟（非刷屏）
- **社群**：每周专题分享+答疑+互动活动
- **1对1**：个性化问题解答+需求挖掘


## 6. 示例
```markdown
## 私域运营方案
### 导流设计
- 诱饵："个人知识体系搭建清单"PDF
- 路径：评论区→私信"知识体系"→自动回复微信号

### 承接流程
1. 添加好友后：发送PDF+自我介绍
2. 第1天：分享一篇知识管理干货
3. 第3天：邀请加入社群

### 核心指标

- 社群活跃度：30%周活跃
- 付费转化率：5%
```
