# 私域运营

## 使用场景

- 小红书粉丝向私域导流
- 私域社群运营
- 用户转化和复购
- 建立长期用户关系

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| current_private_domain.platform | string | wechat / 微信群 |
| current_private_domain.existing_users | number | 现有私域用户数 |
| traffic_source.main_channel | string | 主要流量来源 |
| goal | string | growth / activation / conversion / retention |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| current_private_domain.daily_new | number | 日均新增 |
| traffic_source.daily_views | number | 日曝光 |
| traffic_source.conversion_rate | number | 转化率 |
| resource_available.daily_time | string | 每日可投入时间 |
| resource_available.team_size | number | 团队规模 |

## 执行流程

1. **导流路径设计**：评论区 → 私信 → 微信/社群
2. **诱饵设计**：用户为什么愿意加微信
3. **承接流程**：加好友后的自动化流程
4. **信任建立**：持续提供价值建立信任
5. **分层运营**：按活跃度和付费意愿分层
6. **转化时机**：识别最佳转化节点
7. **社群运营**：社群活跃度和价值交付

### 导流方式
- **评论区引导**："整理了完整版 PDF，需要的扣 1"
- **私信自动回复**：关键词触发资料包
- **主页简介**：微信号/社群入口
- **直播引流**：直播中引导加群

### 私域内容策略
- **朋友圈**：日常分享 + 干货片段 + 生活感悟（非刷屏）
- **社群**：每周专题分享 + 答疑 + 互动活动
- **1 对 1**：个性化问题解答 + 需求挖掘

### 判断标准（if-then）
- 如果导流转化率 <3%，则必须优化诱饵或话术。
- 如果社群周活跃度 <20%，则必须调整内容节奏或互动机制。
- 如果用户 7 天内未互动，则触发一次非销售型价值触达。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 导流设计 | object | 诱饵/路径/话术 |
| 承接流程 | object[] | 添加后第 1/3/7 天动作 |
| 运营日历 | object | 周一/三/五安排 |
| 核心指标 | object | 导流转化率/活跃度/付费转化率 |

## 示例

### 示例 1：知识体系资料包导流
**输入**：
```yaml
current_private_domain:
  platform: "微信"
  existing_users: 200
traffic_source:
  main_channel: "小红书"
  daily_views: 3000
  conversion_rate: 0.02
goal: "growth"
resource_available:
  daily_time: "1小时"
  team_size: 1
```

**输出**：
```markdown
## 私域运营方案
### 导流设计
- 诱饵："个人知识体系搭建清单"PDF
- 路径：评论区 → 私信"知识体系" → 自动回复微信号
- 话术模板："整理了《知识体系搭建清单》，需要的扣 1，我私信发你~"

### 承接流程
1. 添加好友后：发送 PDF + 自我介绍
2. 第1天：分享一篇知识管理干货
3. 第3天：邀请加入社群
4. 第7天：分享训练营预告

### 运营日历
- 周一：社群主题分享「本周一个认知升级」
- 周三：答疑日
- 周五：本周精选内容回顾

### 核心指标
- 导流转化率：8%
- 社群活跃度：30% 周活跃
- 付费转化率：5%
```
