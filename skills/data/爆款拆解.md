# 爆款拆解 Skill

## 1. Skill名称
爆款拆解

## 2. 使用场景
- 发现爆款内容时的深度分析
- 建立爆款模型库
- 内容创作前的参考研究
- 规律总结和方法论提炼

## 3. 输入格式
```yaml
target_post:
  title: ""
  url: ""
  author: ""
  metrics:
    views: 0
    likes: 0
    favorites: 0
    comments: 0
    shares: 0
  publish_time: ""
  content_type: "图文" | "视频"
  niche: "个人成长"
analysis_depth: "quick" | "deep"
```

## 4. 执行流程
1. **标题拆解**：分析标题公式和钩子
2. **封面分析**：视觉吸引力和信息传达
3. **开头拆解**：前3句话如何抓住注意力
4. **结构分析**：内容骨架和节奏
5. **情绪曲线**：读者情绪起伏设计
6. **共鸣点识别**：哪些点引发评论互动
7. **金句提取**：传播性强的句子
8. **模型归纳**：提炼可复用的爆款模型

### 分析维度
- **选题角度**：为什么这个话题会爆？
- **表达方式**：如何让内容有感染力？
- **互动设计**：如何引导用户评论？
- **传播基因**：如何激发转发和收藏？
- **时机因素**：是否踩中热点或趋势？

## 5. 输出格式
```markdown
## 爆款拆解报告
### 基础信息
- 标题：{}
- 数据：{阅读/互动数据}
- 作者：{}
- 发布时间：{}

### 拆解分析
| 维度 | 分析 | 借鉴 |
|------|------|------|
| 标题 | {} | {} |
| 开头 | {} | {} |
| 结构 | {} | {} |
| 情绪 | {} | {} |
| 互动 | {} | {} |

### 爆款模型
{提炼出的可复用模型}

### 我的应用
{如何应用到自己的内容中}
```

## 6. 示例
```markdown
## 爆款拆解报告
### 基础信息
- 标题："30岁才明白的5个人生道理"
- 数据：5.2w赞，1.8w收藏
- 作者：XXX
- 发布时间：周四晚

### 爆款模型
"年龄+数字+人生感悟"公式
- 年龄标签引发精准人群共鸣
- 数字增加预期和可读性
- 人生感悟满足"过来人经验"需求

### 我的应用
创作"28岁产品经理的3个认知升级"
保留年龄+数字结构，替换为个人职业视角
```
