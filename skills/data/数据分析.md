# 数据分析

## 使用场景

- 每周数据复盘 → 含三层内容维度对比
- 内容效果评估 → 按觉察/重构/创造分层分析
- 增长归因分析
- 90 天实验计划进度追踪
- 策略效果量化

## 输入格式

### 必须输入
| 字段 | 类型 | 说明 |
|------|------|------|
| posts_data | object[] | 每篇内容的数据 |
| time_range | string | weekly / monthly / quarterly |

### 可选输入
| 字段 | 类型 | 说明 |
|------|------|------|
| comparison | string | previous_period / same_period_last_month |

### posts_data 字段
| 字段 | 类型 | 说明 |
|------|------|------|
| title | string | 内容标题 |
| publish_date | string | 发布日期 |
| views | number | 阅读量 |
| likes | number | 点赞 |
| favorites | number | 收藏 |
| comments | number | 评论 |
| shares | number | 分享 |
| followers_gained | number | 新增粉丝 |
| content_layer | string | 觉察层 / 重构层 / 创造层 |

## 执行流程

1. **数据清洗**：整理和校验原始数据
2. **核心指标计算**：互动率、转化率、增长率
3. **维度分析**：时间/内容类型/格式多维度
4. **基准对比**：对比历史数据和行业基准
5. **异常识别**：找到表现特别好/差的内容
6. **趋势分析**：识别增长/下降趋势
7. **归因分析**：什么因素驱动了变化
8. **洞察提炼**：转化为可执行的建议

### 核心指标
- **互动率** = (点赞+收藏+评论+分享) / 阅读量
- **转化率** = 粉丝增长 / 阅读量
- **爆款率** = 阅读量 >1 万的内容占比
- **粉丝质量** = 互动粉丝 / 总粉丝
- **内容效率** = 总互动 / 创作时长

### 三层分析维度
每次数据分析增加以下维度：
1. **各层效果对比**：觉察层/重构层/创造层的平均互动率、收藏率、评论率
2. **比例偏差分析**：当前发布比例是否偏离计划？哪一层需要补充？
3. **阶段适应性**：当前数据是否支持进入 90 天实验的下一阶段？

### 判断标准（if-then）
- 如果某篇内容阅读 ≥1 万，则标记为爆款并进入拆解流程。
- 如果互动率低于平均 50%，则标记为低效内容并分析原因。
- 如果三层比例偏离 [[08_数据复盘/90天实验计划.md|90天实验计划]] 超过 15%，则提出调整建议。

## 输出格式

### 必须字段
| 字段 | 类型 | 说明 |
|------|------|------|
| 核心数据 | object | 总阅读/总互动/互动率/粉丝增长 |
| 维度分析 | object | 最佳内容类型/发布时间/标题公式 |
| 三层效果对比 | object | 各层发布数/阅读/互动率 |
| 异常发现 | object | 高光/低效内容及原因 |
| 策略建议 | string[] | 可执行建议 |
| 90天实验检查 | string | 当前阶段/目标/是否达标 |

## 示例

### 示例 1：周度数据分析
**输入**：
```yaml
posts_data:
  - title: "20多岁最怕的不是没钱，是在过二手人生"
    views: 15000
    likes: 1200
    favorites: 800
    comments: 300
    shares: 100
    followers_gained: 120
    content_layer: "觉察层"
time_range: "weekly"
comparison: "previous_period"
```

**输出**：
```markdown
## 数据分析报告
### 核心数据
| 指标 | 本期 | 上期 | 变化 |
|------|------|------|------|
| 总阅读 | 52,300 | 38,100 | +37% |
| 总互动 | 4,180 | 2,850 | +47% |
| 互动率 | 8.0% | 7.5% | +0.5% |
| 粉丝增长 | 320 | 210 | +52% |

### 维度分析
- 最佳内容类型：方法论型（互动率 12%）
- 最佳发布时间：周四 20:00-21:00
- 最佳标题公式：数字+结果型

### 策略建议
1. 增加方法论型内容比例
2. 优化故事型内容的开头钩子
```
