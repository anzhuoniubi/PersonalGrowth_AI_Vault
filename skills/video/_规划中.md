# 视频脚本创作（规划中）

## 状态

本 Skill 处于规划阶段，尚未创建完整实现。当前「半醒之间」IP 以图文内容为主，视频脚本能力待进入下一阶段后补充。

## 何时创建

满足以下任一条件时，由 CEO Agent 决策是否创建：
- 粉丝数 ≥ 5000 且图文内容增长见顶
- Creator 明确要求启动视频内容
- 平台算法明显向视频倾斜，图文流量持续下降 ≥30%

## 创建时应包含

1. **使用场景**：短视频 / 中视频 / 口播 / 分镜脚本
2. **输入格式**：选题卡片、目标时长、平台要求
3. **执行流程**：
   - 钩子设计（前 3 秒）
   - 结构框架（问题-冲突-解决-行动）
   - 口语文感优化
   - 画面/字幕/音效标注
4. **输出格式**：分镜表 + 口播稿 + 发布文案
5. **示例**：真实爆款视频拆解示例
