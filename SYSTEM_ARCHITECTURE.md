## 一、系统定位

**半醒之间 个人 IP 系统** 是一人 AI 自媒体操作系统，
面向"一个普通人在 AI 时代探索个人价值重构"的方向，用 AI Agent 替代传统自媒体团队岗位，
实现从选题、研究、创作、发布、复盘到商业变现的完整闭环。

### 来源
本系统由 ChatGPT 导师对话（15 轮定位梳理）驱动搭建，以 Claude Code 作为系统工程管理器。

## 七、Workflow 自动化流程（5 个）

### 7.1 内容生产流程
> 文件：[[workflows/content_production_workflow.md|content_production_workflow]]

```
输入：一个主题或灵感
    │
    ▼
 0. 内容层定位     ← 判断属于觉察/重构/创造哪一层
    │
    ▼
 1. 用户分析       ← Research Agent + 用户痛点挖掘 Skill
    │
    ▼
 2. 选题优化       ← Topic Agent + 赛道分析 Skill + 三层校准
    │
    ▼
 3. 爆款参考       ← Viral Analysis Agent + 爆款拆解 Skill
    │
    ▼
 4. 内容撰写       ← Content Agent + 标题生成/金句生成/redbook-writing
    │
    ▼
 5. 渲染 Markdown  ← Content Agent + 小红书文案写作 Skill
    │
    ▼
 6. 视觉设计       ← Visual Agent + 封面设计/AI绘图 Skill
    │
    ▼
 7. 图片渲染       ← Content Agent + redbook-auto 工具链
    │
    ▼
 8. 保存归档       ← Knowledge Base Agent → 05_内容生产库/（含三层标签）
    │
    ▼
 9. Creator 审核   ← Creator（人类）
    │
    ▼
10. 发布 + 互动    ← Creator + Content Agent + 互动运营 Skill
    │
    ▼
11. 数据追踪       ← Review Agent + 数据分析 Skill → 10_数据复盘/
    │
    ▼
    复盘报告 → 进入下一轮迭代
```

### 7.2 用户研究工作流
> 文件：[[workflows/research_workflow.md|research_workflow]]
- **频率**：每周 2 次
- **流程**：评论扫描 → 搜索热词采集 → 三层分类 → 用户洞察卡片 → 更新 Memory
- **产出**：评论分析报告、用户洞察卡片、按三层分类的新选题建议

### 7.3 数据复盘工作流
> 文件：[[workflows/review_workflow.md|review_workflow]]
- **频率**：每周日执行
- **流程**：数据整理 → 三层效果对比 → 90 天实验进度检查 → 策略输出 → 归档
- **产出**：周度复盘报告、内容评分卡、下周计划（含三层比例调整）

### 7.4 爆款研究工作流
> 文件：[[workflows/viral_analysis_workflow.md|viral_analysis_workflow]]
- **频率**：每周 2 次
- **流程**：爆款发现（半醒之间视角）→ 深度拆解 → 模型提炼 → 归档更新
- **产出**：爆款拆解报告、爆款模型卡、可迁移选题

### 7.5 商业变现工作流
> 文件：[[workflows/business_workflow.md|business_workflow]]
- **频率**：每月 1 次 + 按需触发
- **当前阶段**：⚪ 监控模式（仅记录洞察，不推动变现）
- **流程**：机会评估 → 产品设计 → 转化路径 → 上线执行 → 复盘优化
- **产出**：产品设计方案、转化漏斗、推广计划

## 八、设计原则

| 原则 | 说明 |
|------|------|
| Agent 负责目标 | 每个 Agent 有明确的角色、目标和产出 |
| Skill 负责能力 | 每个 Skill 是可独立执行的标准化能力单元 |
| Memory 负责长期资产 | 记忆系统积累可持续复用的知识资产 |
| Workflow 负责流程 | 工作流串联 Agent 和 Skill 形成完整闭环 |
| 三层意识 | 所有内容先判断觉察/重构/创造层级，按层调整语气和策略 |
| 单一来源 | 当前阶段三层比例以 [[10_数据复盘/90天实验计划.md|90天实验计划]] 为准，不存数字副本 |

> 🚫 不设计万能 Agent：每个 Agent 职责边界清晰，通过协作而非全能完成复杂任务。
