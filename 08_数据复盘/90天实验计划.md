## 实验总目标

**验证「半醒之间」IP在小红书的可行性。**

具体：
1. 找到用户共鸣最强烈的选题方向
2. 确认三层内容体系是否成立（用户对哪一层反应最好）
3. 积累前1000个真实关注者
4. 形成稳定的内容生产节奏

---

## 阶段划分

### 阶段1：共鸣寻找（第1-30天）

| 项目 | 内容 |
|------|------|
| **目标** | 找到用户共鸣最强烈的选题方向 |
| **发布量** | 30篇 |
| **比例** | 70%觉察 / 17%重构 / 13%创造 |
| **数据目标** | 至少1篇互动率>15%，至少3篇收藏率>8% |
| **关键指标** | 评论深度、收藏/点赞比、高互动选题模式 |
| **关键输出** | 前30篇选题矩阵执行报告 |

### 阶段2：系列化（第31-60天）

| 项目 | 内容 |
|------|------|
| **目标** | 围绕高互动主题形成系列内容 |
| **发布量** | 20篇 |
| **比例** | 50%觉察 / 30%重构 / 20%创造 |
| **数据目标** | 粉丝达到2000+，稳定周互动率>5% |
| **关键指标** | 系列内容的连续互动表现、粉丝增长曲线 |
| **关键输出** | 系列选题方案、账号定位校准报告 |

### 阶段3：行动强化（第61-90天）

| 项目 | 内容 |
|------|------|
| **目标** | 强化"AI+个人品牌实践"标签，让用户看到你在行动 |
| **发布量** | 20篇 |
| **比例** | 40%觉察 / 30%重构 / 30%创造 |
| **数据目标** | 粉丝达到5000+，出现内容被主动转发/讨论 |
| **关键指标** | 粉丝增长加速度、私信/评论中"怎么做"比例 |
| **关键输出** | 90天实验报告、下一阶段战略建议 |

## 关键里程碑

| 里程碑 | 判断标准 | 截止时间 | 状态 |
|--------|---------|---------|:----:|
| M1：找到第一个高互动选题 | 互动率>15%，评论>50条 | 第2周结束 | ⏳ |
| M2：确认三层效果分布 | 至少每层有3篇数据可对比 | 第4周结束 | ⏳ |
| M3：形成第一个系列 | 同一主题连续3篇互动率>8% | 第6周结束 | ⏳ |
| M4：粉丝突破1000 | 真实关注者（非互关） | 第8周结束 | ⏳ |
| M5：粉丝突破5000 | 稳定增长曲线 | 第12周结束 | ⏳ |

## 阶段过渡条件

### 从阶段1→阶段2（第30天检查）
- ✅ 至少1篇爆款（互动率>15%）
- ✅ 已识别出2-3个高互动选题方向
- ✅ 第一组有效数据支持内容层假设
- ❌ 如果以上未达到：延长阶段1最多2周，调整选题策略

### 从阶段2→阶段3（第60天检查）
- ✅ 粉丝≥2000
- ✅ 至少1个系列已经形成
- ✅ 周互动率稳定>5%
- ❌ 如果以上未达到：分析瓶颈，可能调整定位或内容策略

---

## 风险预案

| 风险 | 应对策略 |
|------|---------|
| 30篇发完仍无高互动内容 | 回到"用户研究"，检查是否选错了用户/选题方向 |
| 觉察层有共鸣但重构层无人关注 | 放慢重构层比例，强化觉察层，找到"从觉察到重构"的桥梁选题 |
| 创造层数据始终差 | 检查是否"教"的口气太重，回归"实验者"姿态 |
| 出现意料之外的爆款方向 | 立即追加该方向的系列内容，可能需要调整定位表述 |
| 创作者精力不足，发布节奏掉队 | 减少发布量（每周2篇保底），但保持节奏稳定性 |
