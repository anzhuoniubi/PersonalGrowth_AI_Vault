# 📥 收件箱使用指南

> 位置：00_Inbox/
> 用途：所有新笔记的入口，每日清零

---

## 收件箱规则

任何新笔记必须先进入收件箱，经过处理后再移入对应目录。**禁止直接在目标目录创建文件。**

## 收件箱处理流程

```
1. 新笔记 → 放入 00_Inbox/
    ↓
2. 每日清理 → 检查收件箱所有文件
    ↓
3. 分类打标签
    ↓
4. 移入对应目录（01-13）
    ↓
5. 添加双链关联
```

## 分类映射

| 笔记类型 | → 目标目录 |
|---------|-----------|
| 选题灵感/热点 | 03_选题库 |
| 用户评论/反馈 | 02_用户研究 |
| 知识卡片/读书笔记 | 06_知识体系 |
| 心理学理论/概念 | 07_心理学资料 |
| 案例/参考 | 08_案例库 |
| 提示词/工作流 | 07_提示词库 |
| 商业想法/产品设计 | 09_商业变现 |
| 内容草稿/片段 | 05_内容生产库 |

## 注意事项

- 收件箱文件数**永远不应该超过10个**
- 如果收件箱堆积，优先处理 → 知识库Agent
- 所有文件在移出收件箱前必须补全frontmatter
