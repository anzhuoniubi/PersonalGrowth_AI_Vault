## 流程步骤

### Phase 1: 数据整理（5分钟）
**执行Agent**: 复盘Agent

1. **收集本周数据**
   - 整理所有发布笔记的数据（含三层标注）
   - 收集粉丝增长数据
   - 记录互动数据

2. **数据录入**
   - 填写 [[10_数据复盘/数据看板.md|数据看板]]（含三层维度）
   - 更新 [[10_数据复盘/内容评分卡.md|内容评分卡]]
   - 整理评论数据

### Phase 2: 多维度分析（10分钟）
**执行Agent**: 复盘Agent
**调用 Skill**:
- [[skills/data/数据分析.md|数据分析]]
- [[skills/data/爆款拆解.md|爆款拆解]]
- [[skills/data/A_B测试.md|A/B 测试]]

1. **内容维度分析**
   - 按内容类型对比表现
   - 按发布时间对比效果
   - 按标题公式对比点击率
   - **🔍 新增：按三层维度对比效果** — 哪一层的内容当前表现最好？

2. **三层效果归因**
   - 觉察层：共鸣深度（评论率、收藏比）
   - 重构层：认知价值（收藏率、分享率）
   - 创造层：信任建立（关注转化率、私信）

3. **爆款分析**
   - 识别本周爆款内容
   - 拆解爆款成功因素
   - 与上周爆款对比模式

4. **异常识别**
   - 表现特别好的内容 → 放大策略
   - 表现异常差的内容 → 归因分析

### Phase 3: 策略输出（10分钟）
**执行Agent**: 复盘Agent + CEO Agent

1. **模式总结**
   - 有效策略列表（继续做）
   - 无效策略列表（停止做）
   - 新发现（开始尝试）

2. **策略优化建议**
   - 选题方向调整（含三层比例调整建议）
   - 内容形式优化
   - 发布时间优化
   - 互动策略优化

3. **90天实验进度检查**（新增）
   - 当前处于阶段几？（1-30天/31-60天/61-90天）
   - 发布比例是否符合 [[10_数据复盘/90天实验计划.md|90天实验计划]]？
   - 是否达到阶段目标？→ 是否可以进入下一阶段？

4. **下周计划**
   - 确定下周核心选题（按三层分配）
   - 分配创作优先级
   - 设定关键指标目标

### Phase 4: 归档（5分钟）
**执行Agent**: 知识库Agent

1. **保存复盘报告**
   - 填写 [[10_数据复盘/周度复盘模板.md|周度复盘模板]]
   - 保存到 `10_数据复盘/`

2. **更新选题库**
   - 标记本周选题效果
   - 更新选题优先级

3. **更新 Memory**
   - 如有新发现，更新 [[memory/brand_memory|brand_memory]] / [[memory/audience_memory|audience_memory]]

## 上下游连接

### 上游输入
- 本周发布的所有内容数据
- 评论区互动数据
- 粉丝增长数据

### 下游输出 →
- **[[agents/ceo_agent.md|CEO Agent]]**：周度战略输入
- **[[agents/topic_agent.md|Topic Agent]]**：选题方向调整
- **[[agents/content_agent.md|Content Agent]]**：创作优化指南
- 归档至 `10_数据复盘/`
