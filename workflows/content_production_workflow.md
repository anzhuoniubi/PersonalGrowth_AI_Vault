> 自动化内容生产流程，串联多个Agent和Skill
> 适用于：每日内容创作
> 对齐IP定位：[[01_个人定位/半醒之间IP定位]] | [[06_知识体系/三层内容体系参考]]

## 工作流定义

```yaml
name: 内容生产工作流（半醒之间版）
trigger: Creator输入一个主题
频率: 每日
预估耗时: 30-45分钟
```

## 流程步骤

### Step 0: 内容层定位（新增）
**执行Agent**: CEO Agent / Creator
**判断**: 这个选题属于哪个内容层？
- 🔍 **觉察层**："我为什么活得不像自己？" → 共鸣驱动
- 🔄 **重构层**："AI时代我还能怎么办？" → 认知驱动
- 🛠️ **创造层**："如何把自己变成价值？" → 行动驱动
**输出**: 内容层标注 → 后续所有步骤按该层调整语气和结构

### Step 1: 用户分析
**执行Agent**: 研究Agent
**调用Skill**: skills/research/用户痛点挖掘.md
**输入**: 主题 + 当前用户画像 → [[02_用户研究/核心用户画像]]
**输出**: 用户洞察卡片 → 保存到 02_用户研究/
**预计耗时**: 3分钟

### Step 2: 选题优化 & 三层校准
**执行Agent**: 选题Agent
**调用Skill**: skills/strategy/赛道分析.md
**输入**: 主题 + 热点数据 + 爆款参考 + 三层标注
**输出**: 选题评分卡（含三层对标） → 保存到 03_选题库/
**检查**: 当前选题池各层比例是否合理？（当前阶段：70%觉察/20%重构/10%创造）
**预计耗时**: 3分钟

### Step 3: 爆款参考
**执行Agent**: 选题Agent
**调用Skill**: skills/data/爆款拆解.md
**输入**: 同类爆款内容URL
**输出**: 爆款拆解报告 → 保存到 04_爆款拆解库/
**预计耗时**: 5分钟

### Step 4: 内容撰写
**执行Agent**: 内容Agent
**调用Skill**: skills/content/小红书文案写作.md（Phase 1）
**输入**: 选题 + 标题方案 + 内容层标注
**原则**:
- **先解决痛苦，再升维**
- **三层语气适配**：觉察层温柔共鸣 / 重构层理性启发 / 创造层真实记录
- **探索者语气**：不是"我教你"，是"我在做，你看"
**输出**: 完整正文 + 标题 + 标签 + 金句
**预计耗时**: 10分钟

### Step 5: 渲染Markdown生成
**执行Agent**: 内容Agent
**调用**: skills/content/小红书文案写作.md（Phase 2）
**操作**: 将正文转换为渲染专用Markdown（含frontmatter）
**输出**: content.md
**预计耗时**: 2分钟

### Step 6: 图片渲染
**执行Agent**: 内容Agent
**调用Script**: skills/redbook-auto/scripts/render_xhs.py
**操作**: 按内容层选主题（觉察-sketch / 重构-default / 创造-botanical），默认auto-split
**输出**: cover.png（封面）+ card_N.png（正文卡片）
**预计耗时**: 3分钟

### Step 7: 保存归档
**执行Agent**: 知识库Agent
**输入**: 完整内容包（含渲染图片引用）
**输出**: 保存到 05_内容生产库/{YYYY-MM-DD}_{标题}.md
**预计耗时**: 2分钟

### Step 8: Creator审核
**执行人**: Creator
**检查项**:
- [ ] 内容真实性
- [ ] 风格一致性
- [ ] 渲染效果
- [ ] 合规检查
**预计耗时**: 5分钟

### Step 9: 发布（可选）
**执行Agent**: 内容Agent / Creator
**调用Script**: skills/redbook-auto/scripts/publish_xhs.py
**流程**: 默认仅自己可见 → 确认后 --public 公开发布
**预计耗时**: 2分钟

### Step 10: 数据追踪
**执行Agent**: 复盘Agent
**调用Skill**: skills/data/数据分析.md
**触发**: 发布后24小时 + 72小时
**输出**: 数据追踪记录 → 10_数据复盘/
**预计耗时**: 每轮2分钟

### Step 6: 视觉设计
**执行Agent**: 视觉Agent ✅（已激活）
**调用Skill**: skills/visual/封面设计.md + skills/visual/AI绘图提示词.md
**输入**: 已定标题 + 正文核心主张 + 情绪基调
**输出**: 封面方案（含三区布局+配色+字体）+ AI绘图提示词 → 追加到内容文件
**预计耗时**: 5分钟

#### ❖ 实际执行流程（基于已发布内容验证）

```
Step 6a: 内容分析
  └─ 提取核心表达点、内容类型(认知观点/情绪共鸣)、情绪基调
Step 6b: 选择构图
  └─ 决定构图类型（大字报/三阶式/上下结构/Cover式）
Step 6c: 三区布局
  └─ 上区主标题 → 中区视觉主体 → 下区辅助信息
Step 6d: 配色方案
  └─ 主色(暗夜灰蓝) + 文字色(白) + 点缀色(暖金)
Step 6e: AI绘图提示词 → 备用纯文字排版方案
Step 6f: 小图测试（✓ 手机屏可读 / ✓ 3秒理解主张 / ✓ 无遮挡 / ✓ 无违禁词）
```

#### ❖ 已验证的封面模板库（来自实际发布）

| 类型 | 适用场景 | 已用案例 | 效果 |
|------|---------|---------|:----:|
| 大字报 | 强主张/反差型标题 | 认知反差类 | CTR 8.6% |
| 三阶式 | 场景代入+标签 | 社会时钟觉醒类 | CTR 4.2% |
| 上下结构 | 情绪共鸣型 | 身份标签类 | CTR 19.5% 🔥 |

> ℹ️ **经验**：CTR与封面精致度关系不大，与标题的情绪触发强度正相关。
> 高CTR的关键在于标题的争议性或情绪冲击力，而非封面视觉本身。
> 但封面必须保证：小图可读、3秒理解主张、无违禁词。

### Step 7: 保存归档
**执行Agent**: 知识库Agent ✅（已激活）
**输入**: 完整内容包（标题/正文/封面方案/金句/标签）
**输出**: 内容包保存到 05_内容生产库/{YYYY-MM-DD}_{标题}.md
**预计耗时**: 2分钟

#### ❖ 归档规范（已落地）
- 文件名：`YYYY-MM-DD_{内容关键词}.md`
- Frontmatter：title / date / status(已发布/待发布/草稿) / source / agent / tags
- 正文后依次追加：标题区 → 封面方案 → 正文 → 标签 → 金句 → 敏感词自查 → 黄金三角质检 → 创作说明 → **数据追踪表**（72h内补充）
- 每次用Edit追加，不用Write覆盖全文件

### Step 8: Creator审核
**执行人**: Creator（人类）
**检查项**:
- [ ] 内容真实性：是否有亲身经历支撑
- [ ] 风格一致性：是否符合个人表达风格
- [ ] 信息准确性：观点和事实是否正确
- [ ] 合规检查：是否违反平台规则
- [ ] 价值判断：是否对目标用户有帮助
**预计耗时**: 5分钟

### Step 9: 发布 + 互动
**执行人**: Creator + 互动运营Skill
**调用Skill**: skills/growth/互动运营.md
**输出**: 已发布内容 + 评论区运营
**预计耗时**: 5分钟

### Step 10: 数据追踪
**执行Agent**: 复盘Agent
**调用Skill**: skills/data/数据分析.md
**触发**: 发布后24小时 + 72小时
**输出**: 数据追踪记录 → 保存到 10_数据复盘/
**预计耗时**: 每轮2分钟

## 检查清单（供Creator使用）

### 创作前
- [ ] 我从哪个Skill开始？
- [ ] 今天的目标用户群体是谁？
- [ ] 这个选题是否在选题库中评分通过？
- [ ] 核心主张是什么？一句话能说清吗？

### 创作中
- [ ] 是否参考了记忆系统中的品牌和风格指南？
- [ ] 标题是否符合标题公式？字数≤20字？
- [ ] 开头是否有钩子？
- [ ] 正文是否有结构？（SCQA/故事/清单）
- [ ] 金句是否单独提取（≥3句）？
- [ ] 结尾是否有行动号召？
- [ ] 封面方案是否已确认（构图类型+配色+备选）？

### 发布前
- [ ] 是否经过真实性审核？
- [ ] 封面在小图下是否可读？3秒能理解主张？
- [ ] 敏感词自查是否完成？
- [ ] 黄金三角质检是否通过？
- [ ] 标签是否覆盖核心关键词？
- [ ] 是否准备了互动回复模板？

### 发布后
- [ ] 是否更新了选题库状态（已发布+勾选数据复盘待做）？
- [ ] 是否在1小时内回复评论？
- [ ] 24h数据是否已填入数据追踪表？
- [ ] 72h复盘是否在窗口期内完成？
- [ ] GEPA数据集是否已追加记录？
- [ ] 周报中是否包含这篇内容？
